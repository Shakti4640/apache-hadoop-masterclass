PROJECT 36: BULK LOADING 500 MILLION RECORDS INTO HBASE WITHOUT KILLING IT

🔴 HYPOTHETICAL PROBLEM

SCENARIO:
→ You are the same architect from Project 35 (Smart City IoT Platform)
→ The HBase table is live — serving real-time queries from 500K sensors
→ Now management wants 3 YEARS of historical data loaded
→ Historical data sits in HDFS as binary sensor files (Project 32 format)
→ Volume: 500 million records across 50,000 files

THE NAIVE APPROACH (what a junior engineer tried):

  "I'll just write a MapReduce job that reads HDFS files
   and does HBase Puts from each mapper."

  WHAT HAPPENED:
  
  Hour 1:
  → 200 mappers each doing 1000 Puts/sec to HBase
  → 200,000 Puts/sec hitting the cluster
  → Region servers MemStore fills up rapidly
  → Constant flushes: MemStore → HFile every few seconds
  
  Hour 2:
  → Each region now has 50+ HFiles
  → Compaction kicks in → reading + merging + writing HFiles
  → CPU at 100% on all region servers (compaction + serving)
  → Read latency spikes from 10ms to 5 SECONDS
  → Real-time dashboard goes RED — live queries timing out
  
  Hour 3:
  → MemStore cannot flush fast enough (compaction blocking)
  → RegionServer throws: "MemStore size exceeds upper limit"
  → Region servers start blocking ALL writes
  → Some region servers OOM → crash → region reassignment
  → Cluster enters "compaction death spiral"
  
  Hour 4:
  → Only 50 million records loaded (10% done)
  → Cluster is unusable — both reads AND writes failing
  → Engineers kill the load job
  → Cluster takes 6 HOURS to recover from compaction backlog
  → Management is furious — dashboard was down during business hours

  TOTAL DAMAGE:
  → 10 hours wasted
  → Only 10% loaded
  → Production service disrupted
  → Need to restart from scratch

THE QUESTION:
→ How do you load 500 million records into a LIVE HBase table
  without disrupting real-time queries?
→ How do you bypass the write path entirely?
→ How do you generate pre-sorted HFiles that HBase can
  adopt directly without compaction storms?

🔍 HOW TO APPROACH IT

STEP 1: Understand why direct Puts fail at scale

  THE HBASE WRITE PATH (normal operation):

  ┌─────────────────────────────────────────────────────────┐
  │  Client Put("row_key", "cf:col", "value")               │
  │       │                                                  │
  │       ▼                                                  │
  │  RegionServer receives Put                               │
  │       │                                                  │
  │       ├──→ Write to WAL (Write-Ahead Log) on disk        │
  │       │    → Sequential append → fast                    │
  │       │    → Ensures durability if server crashes         │
  │       │                                                  │
  │       └──→ Write to MemStore (in-memory sorted buffer)   │
  │            → Red-black tree or skip list                  │
  │            → Sorted by row key                           │
  │            │                                              │
  │            ▼ (when MemStore reaches flush threshold)      │
  │       Flush MemStore → HFile on HDFS                     │
  │            → Creates a new immutable sorted file          │
  │            → Each flush = 1 new HFile                    │
  │            │                                              │
  │            ▼ (when too many HFiles accumulate)            │
  │       Compaction: merge multiple HFiles → fewer HFiles   │
  │            → Minor: merge small files                    │
  │            → Major: merge ALL files → 1 file             │
  │            → VERY expensive: reads all data + writes all │
  │                                                          │
  └─────────────────────────────────────────────────────────┘

  AT 200K PUTS/SEC:
  → MemStore flushes every ~2 seconds (128MB at ~64 bytes/Put)
  → Creates 30 HFiles per minute per region
  → Compaction cannot keep up
  → HFile count grows unbounded → read performance degrades
  → Eventually: "too many store files" error blocks writes

STEP 2: Understand the Bulk Load alternative

  BULK LOAD BYPASSES THE ENTIRE WRITE PATH:

  ┌─────────────────────────────────────────────────────────┐
  │  BULK LOAD PATH (what we will build)                     │
  │                                                          │
  │  Step 1: MapReduce job reads source data from HDFS       │
  │          → Uses our custom InputFormat (Project 32)      │
  │          → Mapper emits (row_key, KeyValue) pairs        │
  │                                                          │
  │  Step 2: Reducer sorts by row key                        │
  │          → HFileOutputFormat2 writes sorted HFiles       │
  │          → One HFile per region per column family         │
  │          → Files are written to a TEMP directory on HDFS  │
  │                                                          │
  │  Step 3: LoadIncrementalHFiles (completebulkload)        │
  │          → Moves HFiles into HBase region directories     │
  │          → Atomic operation — region picks up new files   │
  │          → NO WAL writes, NO MemStore, NO compaction      │
  │          → Region server doesn't even notice until next   │
  │            read hits the new file                         │
  │                                                          │
  │  RESULT:                                                  │
  │  → Zero impact on live read/write traffic                 │
  │  → No compaction storms                                   │
  │  → No MemStore pressure                                   │
  │  → Load completes in hours, not days                      │
  │  → Cluster remains fully operational throughout           │
  │                                                          │
  └─────────────────────────────────────────────────────────┘

  DIRECT PUTS:

✅ WHAT SHOULD DO

1. USE HFileOutputFormat2.configureIncrementalLoad()
   → This single method call configures EVERYTHING:
     → Sets the correct partitioner (matches region boundaries)
     → Sets the correct reducer (HFile writer)
     → Sets the correct output key/value classes
     → Sets the correct comparator (KeyValue sorting)
     → Sets the number of reducers (one per region)
   → You ONLY need to write the Mapper

2. SORT KEYVALUES WITHIN EACH ROW
   → HFile requires KeyValues in EXACT sorted order:
     → First by row key
     → Then by column family
     → Then by column qualifier
     → Then by timestamp (descending)
   → If you emit multiple columns per row: emit them in order
   → HFileOutputFormat2 handles the cross-row sorting

3. PRE-CREATE THE TABLE BEFORE BULK LOAD
   → Table must exist with correct schema
   → Region split points must be defined (Project 35 pre-splitting)
   → Bulk load tool uses existing region boundaries for partitioning

4. USE A TEMPORARY HDFS DIRECTORY FOR HFILE OUTPUT
   → Don't write HFiles directly into HBase directory
   → Write to /tmp/bulk_load/ or similar
   → completebulkload MOVES files (fast rename, not copy)

5. VALIDATE ROW COUNT BEFORE AND AFTER
   → Count input records (MapReduce counters)
   → Count HBase rows after load (scan with counter)
   → Must match — if not, investigate data loss

6. RUN DURING OFF-PEAK HOURS (even though impact is minimal)
   → The MapReduce job uses cluster compute resources
   → completebulkload is fast but briefly locks regions
   → Best practice: run overnight even though it's safe during the day

🚫 WHAT SHOULD NOT DO

1. DO NOT EMIT UNSORTED KEYVALUES FROM MAPPER
   → HFile is a sorted data structure — cannot contain unsorted data
   → If mapper emits columns out of order for same row key:
     → HFileOutputFormat2 throws: "Added a key not lexically larger"
   → SOLUTION: Emit columns in alphabetical order per row

2. DO NOT SKIP THE PARTITIONER CONFIGURATION
   → Without proper partitioning: all data goes to one reducer
   → One reducer writes one giant HFile → cannot be loaded
   → HFileOutputFormat2.configureIncrementalLoad() handles this

3. DO NOT LOAD HFILES FROM A DIFFERENT TABLE SCHEMA
   → HFile column families must match table's column families
   → If HFile has CF "data" but table has CF "d" → load fails
   → Column qualifiers can be anything (dynamic in HBase)

4. DO NOT DELETE HFILES BEFORE CONFIRMING LOAD SUCCESS
   → Keep temp directory until verified
   → If load fails midway: HFiles still exist for retry
   → Only clean up after row count validation passes

5. DO NOT BULK LOAD DURING MAJOR COMPACTION
   → If a region is undergoing major compaction
   → New HFile added by bulk load may be immediately compacted
   → Not harmful but wastes I/O
   → Disable auto-compaction during load, re-enable after

6. DO NOT GENERATE HFILES LARGER THAN REGION MAX SIZE
   → Default region max: 10GB
   → If one HFile > 10GB → triggers immediate region split
   → Split during load adds complexity
   → SOLUTION: With 256 regions and 500M records:
     → ~2M records per region → ~100-200MB per HFile → well under limit

7. DO NOT USE KeyValue DIRECTLY IN NEWER HBASE VERSIONS
   → HBase 2.x deprecates KeyValue in favor of ExtendedCell
   → Use CellUtil.createCell() or KeyValueUtil for compatibility
   → Our code handles both for portability

   → Bulk load is a BATCH operation
   → Cannot bulk-load one record at a time
   → Use buffered Puts (Project 35 SensorDataWriter) for streaming
   → Bulk load is for periodic large batches

2. WHEN DATA IS SMALL (< 10,000 rows)
   → MapReduce job startup overhead: ~30 seconds
   → Direct Puts for 10K rows: ~1 second
   → Bulk load is for millions+ of rows

3. WHEN TABLE SCHEMA IS CHANGING
   → HFiles are tied to column family structure
   → If you're adding/removing CFs: alter table first, then bulk load

4. WHEN YOU NEED IMMEDIATE VISIBILITY
   → Bulk load has a delay: MR job time + load time
   → If data must be queryable within seconds of arrival
   → Use direct Puts for low-latency visibility

═══════════════════════════════════════════════════════════════
MECHANIC 2: HOW configureIncrementalLoad() WORKS
═══════════════════════════════════════════════════════════════

  When you call:
    HFileOutputFormat2.configureIncrementalLoad(job, table, locator)

  It does ALL of the following automatically:

  1. READS REGION BOUNDARIES from HBase table metadata
     → Gets start key of each region
     → Creates a partition file listing all split points

  2. SETS TotalOrderPartitioner
     → Each mapper output is routed to correct reducer
     → Reducer N writes HFile for Region N
     → Number of reducers = number of regions

  3. SETS OUTPUT FORMAT to HFileOutputFormat2
     → Reducer receives sorted KeyValues
     → Writes them as HFile blocks with indexes and bloom filters
     → Compression applied per configuration

  4. SETS KEY/VALUE CLASSES
     → Output key: ImmutableBytesWritable (row key)
     → Output value: KeyValue (or Cell)

  5. SETS COMPARATOR
     → KeyValue.KVComparator for correct sort order
     → Ensures row key → CF → qualifier → timestamp ordering

  RESULT:
  → Developer writes ONLY the mapper
  → Everything else is automatic

═══════════════════════════════════════════════════════════════
MECHANIC 3: HOW completebulkload WORKS
═══════════════════════════════════════════════════════════════

  LoadIncrementalHFiles (completebulkload) tool:

  Step 1: Scan the output directory for HFiles
    /tmp/bulk_load/d/          ← column family "d"
      ├── 0000000000           ← HFile for Region 0
      ├── 0000000001           ← HFile for Region 1
      └── ...

  Step 2: For each HFile:
    a. Read the first and last key from the file
    b. Determine which region(s) this file belongs to
    c. If file fits within ONE region:
       → MOVE (HDFS rename) file into region's store directory
       → /hbase/data/default/sensor_readings/<region_hash>/d/
       → Rename is O(1) in HDFS — instantaneous
    d. If file SPANS multiple regions (region split occurred):
       → SPLIT the HFile at the region boundary
       → Load each piece into its respective region

  Step 3: Notify each affected RegionServer
    → "Hey, Region X has a new store file"
    → RegionServer updates its in-memory file list
    → Next read on that region sees the new data

  Step 4: Verify all HFiles were loaded
    → Reports success/failure per file
    → Failed files can be retried

  THE KEY INSIGHT:
  → HFile is ALREADY in the correct format for HBase to read
  → No conversion, no parsing, no sorting needed
  → Just a file rename in HDFS → data is immediately available
  → This is why it's called "bulk LOAD" not "bulk INSERT"

═══════════════════════════════════════════════════════════════
MECHANIC 4: KEYVALUE SORT ORDER WITHIN A ROW
═══════════════════════════════════════════════════════════════

  For row key 0x4B_A7F3219C_7FFE96C8A25C03FF:
  
  Mapper must emit KeyValues in THIS order:
  
  1. (row, d:aq, 1705312800000, value)    ← "aq" < "h" < "nl" < "p" ...
  2. (row, d:h,  1705312800000, value)
  3. (row, d:nl, 1705312800000, value)
  4. (row, d:p,  1705312800000, value)
  5. (row, d:s,  1705312800000, value)
  6. (row, d:sid, 1705312800000, value)
  7. (row, d:t,  1705312800000, value)
  
  If metadata column family also present:
  8. (row, m:type, 1705312800000, value)   ← CF "m" > CF "d"
  9. (row, m:zone, 1705312800000, value)
  
  SORT HIERARCHY:
  row_key (ascending)
    → column_family (ascending byte order)
      → qualifier (ascending byte order)
        → timestamp (DESCENDING — newest first)

  IF YOU EMIT OUT OF ORDER:
  → "Added a key not lexically larger than previous"
  → Job fails immediately
  → SOLUTION: Use TreeMap to sort columns before emitting

⚠️ COMMON PITFALLS AND FAILURE MODES

PITFALL 1: "Added a key not lexically larger than previous"
→ CAUSE: KeyValues emitted out of sort order
→ Most common: columns not alphabetically sorted within a row
→ Or: two mappers emit same row key → same reducer → duplicate keys
→ SOLUTION: Sort column qualifiers before emitting
→ Use TreeMap<byte[], byte[]> for column ordering

PITFALL 2: "completebulkload splits every HFile"
→ CAUSE: Table regions changed between HFile generation and loading
→ New regions were created (auto-split) during the MR job
→ Each HFile now spans 2+ regions → tool must split them
→ SOLUTION: Disable auto-splitting during bulk load
→ Or: accept the splitting — it works, just slower

PITFALL 3: "HFile generation succeeded but load says 'no files'"
→ CAUSE: Output directory structure wrong
→ HFiles must be in subdirectories named by column family
→ Expected: /tmp/bulk_load/d/00000  (not /tmp/bulk_load/00000)
→ HFileOutputFormat2 creates correct structure automatically
→ Don't restructure the output manually

PITFALL 4: "Bulk load completed but data not visible"
→ CAUSE: Client cache has stale region locations
→ HBase client caches region → server mapping
→ After bulk load: new data in regions may need cache refresh
→ SOLUTION: Close and reopen Connection, or wait for cache expiry

PITFALL 5: "Permission denied during completebulkload"
→ CAUSE: HFiles created by MR user, but HBase runs as hbase user
→ HBase cannot read/move files owned by another user
→ SOLUTION: Set proper permissions on HFile output directory
→ hadoop fs -chmod -R 777 /tmp/bulk_load/
→ Or: run MR job as hbase user
→ Or: set hbase.bulkload.staging.dir with correct permissions

PITFALL 6: "Job runs but produces 0 HFiles"
→ CAUSE: Mapper emits wrong key type
→ HFileOutputFormat2 expects ImmutableBytesWritable as key
→ If you emit Text or LongWritable → silently ignored
→ SOLUTION: Verify mapper output types match configuration

📈 SCALABILITY CONSIDERATIONS

1. HFILE SIZE MANAGEMENT

   500M records across 256 regions:
   → ~1.95M records per region
   → Each record: ~200 bytes (key + columns)
   → Per region HFile: ~390MB raw, ~100MB compressed
   → WELL within region size limits (10GB max)

   At larger scale (10 billion records):
   → ~39M records per region
   → Per region HFile: ~7.8GB raw, ~2GB compressed
   → Still OK, but approaching limits
   → Consider more regions (512 or 1024 pre-splits)

2. MAPREDUCE JOB SIZING

   ┌────────────────────┬────────────────┬──────────────────┐
   │ INPUT VOLUME       │ MAPPERS        │ REDUCERS          │
   ├────────────────────┼────────────────┼──────────────────┤
   │ 50GB (500M rows)   │ ~400 (128MB/split) │ 256 (1/region)│
   │ 500GB (5B rows)    │ ~4,000         │ 256              │
   │ 5TB (50B rows)     │ ~40,000        │ 512+             │
   └────────────────────┴────────────────┴──────────────────┘
   
   BOTTLENECK: Reducers write HFiles → disk I/O bound
   → 256 reducers writing simultaneously = ~256 × 100MB/s = 25GB/s
   → Limited by HDFS write bandwidth
   → On 50-node cluster: ~500MB/s per node = 25GB/s → matches

   ┌────────────────────┬──────────────────┬───────────────┐
   │ RECORDS            │ MR JOB TIME      │ LOAD TIME     │
   ├────────────────────┼──────────────────┼───────────────┤
   │ 10 million         │ ~5 minutes       │ ~30 seconds   │
   │ 100 million        │ ~30 minutes      │ ~2 minutes    │
   │ 500 million        │ ~2 hours         │ ~5 minutes    │
   │ 5 billion          │ ~15 hours        │ ~30 minutes   │
   └────────────────────┴──────────────────┴───────────────┘
   
   LOAD TIME is fast because it's just HDFS renames
   MR JOB TIME is the bottleneck — standard MapReduce scaling applies

package com.smartcity.bulkload;

import com.smartcity.hbase.SensorRowKey;
import com.smartcity.hbase.SensorTableManager;

import org.apache.hadoop.hbase.KeyValue;
import org.apache.hadoop.hbase.io.ImmutableBytesWritable;
import org.apache.hadoop.hbase.util.Bytes;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Mapper;

import java.io.IOException;
import java.util.TreeMap;

public class BulkLoadMapper
        extends Mapper<LongWritable, Text, ImmutableBytesWritable, KeyValue> {

    // Output key wrapper — reused to reduce GC pressure
    private ImmutableBytesWritable outputKey = new ImmutableBytesWritable();

    enum BulkLoadCounters {
        INPUT_RECORDS_READ,
        KEYVALUES_EMITTED,
        RECORDS_SKIPPED_INVALID,
        RECORDS_SKIPPED_PARSE_ERROR
    }

    @Override
    protected void map(LongWritable offset, Text value, Context context)
            throws IOException, InterruptedException {

        context.getCounter(BulkLoadCounters.INPUT_RECORDS_READ).increment(1);

        String line = value.toString();
        String[] parts = line.split("\t");

        // Expected format (from pre-processed CSV or text conversion):
        // sensor_id \t timestamp \t temperature \t humidity \t pressure \t air_quality \t noise_level \t status
        if (parts.length < 8) {
            context.getCounter(BulkLoadCounters.RECORDS_SKIPPED_INVALID).increment(1);
            return;
        }

        try {
            String sensorId = parts[0].trim();
            long timestamp = Long.parseLong(parts[1].trim());
            float temperature = Float.parseFloat(parts[2].trim());
            float humidity = Float.parseFloat(parts[3].trim());
            float pressure = Float.parseFloat(parts[4].trim());
            int airQuality = Integer.parseInt(parts[5].trim());
            float noiseLevel = Float.parseFloat(parts[6].trim());
            byte status = Byte.parseByte(parts[7].trim());

            // ── Build row key using Project 35 design ──
            byte[] rowKey = SensorRowKey.createRowKey(sensorId, timestamp);
            outputKey.set(rowKey);

            // ══════════════════════════════════════════════════════
            // CRITICAL: Emit KeyValues in SORTED ORDER per row
            //
            // Sort hierarchy: row_key → column_family → qualifier
            // Column family "d" columns in alphabetical order:
            //   aq, h, nl, p, s, sid, t
            //
            // Using TreeMap to guarantee alphabetical ordering
            // ══════════════════════════════════════════════════════

            TreeMap<byte[], byte[]> columns = new TreeMap<>(Bytes.BYTES_COMPARATOR);
            columns.put(SensorTableManager.COL_AIR_QUALITY, Bytes.toBytes(airQuality));
            columns.put(SensorTableManager.COL_HUMIDITY, Bytes.toBytes(humidity));
            columns.put(SensorTableManager.COL_NOISE_LEVEL, Bytes.toBytes(noiseLevel));
            columns.put(SensorTableManager.COL_PRESSURE, Bytes.toBytes(pressure));
            columns.put(SensorTableManager.COL_STATUS, new byte[]{status});
            columns.put(SensorTableManager.COL_SENSOR_ID, Bytes.toBytes(sensorId));
            columns.put(SensorTableManager.COL_TEMPERATURE, Bytes.toBytes(temperature));

            // ── Emit each column as a separate KeyValue ──
            for (java.util.Map.Entry<byte[], byte[]> entry : columns.entrySet()) {
                KeyValue kv = new KeyValue(
                    rowKey,                        // row key
                    SensorTableManager.CF_DATA,    // column family
                    entry.getKey(),                // column qualifier
                    timestamp,                     // cell timestamp
                    entry.getValue()               // cell value
                );

                context.write(outputKey, kv);
                context.getCounter(BulkLoadCounters.KEYVALUES_EMITTED).increment(1);
            }

        } catch (NumberFormatException e) {
            context.getCounter(BulkLoadCounters.RECORDS_SKIPPED_PARSE_ERROR).increment(1);
        }
    }
}

package com.smartcity.bulkload;

import com.smartcity.hbase.SensorTableManager;

import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.conf.Configured;

import org.apache.hadoop.hbase.client.*;
import org.apache.hadoop.hbase.io.ImmutableBytesWritable;
import org.apache.hadoop.hbase.mapreduce.HFileOutputFormat2;
import org.apache.hadoop.hbase.tool.BulkLoadHFilesTool;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.input.TextInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;
import org.apache.hadoop.util.Tool;
import org.apache.hadoop.util.ToolRunner;

import java.io.IOException;

public class BulkLoadDriver extends Configured implements Tool {

    @Override
    public int run(String[] args) throws Exception {

        if (args.length < 2) {
            System.err.println("Usage: BulkLoadDriver <input-path> <temp-hfile-path>");
            System.err.println("  input-path:     HDFS path to sensor data (text/csv format)");
            System.err.println("  temp-hfile-path: HDFS temp directory for generated HFiles");
            return 1;
        }

        String inputPath = args[0];
        String hfileOutputPath = args[1];

        Configuration conf = HBaseConfiguration.create(getConf());
        FileSystem fs = FileSystem.get(conf);

        // ── Clean up previous run ──
        Path outputDir = new Path(hfileOutputPath);
        if (fs.exists(outputDir)) {
            System.out.println("  Cleaning previous HFile output...");
            fs.delete(outputDir, true);
        }

        System.out.println("═══════════════════════════════════════════════════════");
        System.out.println("  HBASE BULK LOAD PIPELINE");
        System.out.println("═══════════════════════════════════════════════════════");
        System.out.printf("  Input:    %s%n", inputPath);
        System.out.printf("  HFiles:   %s%n", hfileOutputPath);
        System.out.printf("  Table:    %s%n", SensorTableManager.TABLE_NAME);

        // ══════════════════════════════════════════
        // PHASE 1: Generate HFiles via MapReduce
        // ══════════════════════════════════════════

        System.out.println("\n  ┌──────────────────────────────────────┐");
        System.out.println("  │  PHASE 1: GENERATE HFILES             │");
        System.out.println("  └──────────────────────────────────────┘\n");

        long phase1Start = System.currentTimeMillis();

        try (Connection connection = ConnectionFactory.createConnection(conf)) {
            Table table = connection.getTable(SensorTableManager.TABLE_NAME);
            RegionLocator locator = connection.getRegionLocator(SensorTableManager.TABLE_NAME);

            Job job = Job.getInstance(conf, "HBase Bulk Load - Generate HFiles");
            job.setJarByClass(BulkLoadDriver.class);

            // ── Input configuration ──
            job.setInputFormatClass(TextInputFormat.class);
            FileInputFormat.addInputPath(job, new Path(inputPath));

            // ── Mapper configuration ──
            job.setMapperClass(BulkLoadMapper.class);
            job.setMapOutputKeyClass(ImmutableBytesWritable.class);
            job.setMapOutputValueClass(KeyValue.class);

            // ── Output path ──
            FileOutputFormat.setOutputPath(job, outputDir);

            // ══════════════════════════════════════════════════════
            // THE MAGIC LINE: Configure everything for HFile generation
            //
            // This single call:
            // → Sets TotalOrderPartitioner (routes keys to correct reducer)
            // → Sets number of reducers (one per region)
            // → Sets HFileOutputFormat2 as output format
            // → Sets correct comparator for KeyValue sorting
            // → Sets compression to match table configuration
            // → Sets bloom filter type to match table configuration
            // ══════════════════════════════════════════════════════
            HFileOutputFormat2.configureIncrementalLoad(job, table, locator);

            System.out.println("  Configured incremental load:");
            System.out.printf("    Regions: %d%n", locator.getAllRegionLocations().size());
            System.out.printf("    Reducers: %d%n", job.getNumReduceTasks());
            System.out.println("    Starting MapReduce job...\n");

            boolean success = job.waitForCompletion(true);

            if (!success) {
                System.err.println("  🔴 HFile generation FAILED");
                return 1;
            }

            long phase1Time = (System.currentTimeMillis() - phase1Start) / 1000;

            // ── Print Phase 1 metrics ──
            long inputRecords = job.getCounters()
                .findCounter(BulkLoadMapper.BulkLoadCounters.INPUT_RECORDS_READ).getValue();
            long kvEmitted = job.getCounters()
                .findCounter(BulkLoadMapper.BulkLoadCounters.KEYVALUES_EMITTED).getValue();
            long skippedInvalid = job.getCounters()
                .findCounter(BulkLoadMapper.BulkLoadCounters.RECORDS_SKIPPED_INVALID).getValue();
            long skippedParse = job.getCounters()
                .findCounter(BulkLoadMapper.BulkLoadCounters.RECORDS_SKIPPED_PARSE_ERROR).getValue();

            System.out.println("\n  ✅ Phase 1 COMPLETE");
            System.out.printf("    Time:             %d seconds (%.1f minutes)%n",
                phase1Time, phase1Time / 60.0);
            System.out.printf("    Input records:    %,d%n", inputRecords);
            System.out.printf("    KeyValues emitted: %,d%n", kvEmitted);
            System.out.printf("    Skipped (invalid): %,d%n", skippedInvalid);
            System.out.printf("    Skipped (parse):   %,d%n", skippedParse);
            System.out.printf("    KVs per record:   %.1f%n",
                (double) kvEmitted / Math.max(1, inputRecords - skippedInvalid - skippedParse));

            // ── Verify HFile output ──
            System.out.println("\n  Generated HFiles:");
            org.apache.hadoop.fs.FileStatus[] hfiles = fs.listStatus(
                new Path(hfileOutputPath + "/" + 
                    org.apache.hadoop.hbase.util.Bytes.toString(SensorTableManager.CF_DATA)));
            if (hfiles != null) {
                long totalHFileSize = 0;
                for (org.apache.hadoop.fs.FileStatus hf : hfiles) {
                    totalHFileSize += hf.getLen();
                }
                System.out.printf("    Count: %d HFiles%n", hfiles.length);
                System.out.printf("    Total size: %.2f GB%n",
                    totalHFileSize / (1024.0 * 1024.0 * 1024.0));
                System.out.printf("    Avg size: %.2f MB%n",
                    totalHFileSize / (1024.0 * 1024.0 * Math.max(1, hfiles.length)));
            }

            table.close();
            locator.close();
        }

        // ══════════════════════════════════════════
        // PHASE 2: Load HFiles into HBase
        // ══════════════════════════════════════════

        System.out.println("\n  ┌──────────────────────────────────────┐");
        System.out.println("  │  PHASE 2: LOAD HFILES INTO HBASE      │");
        System.out.println("  └──────────────────────────────────────┘\n");

        long phase2Start = System.currentTimeMillis();

        // ── Set permissions so HBase can read the files ──
        System.out.println("  Setting file permissions...");
        fs.setPermission(outputDir, 
            new org.apache.hadoop.fs.permission.FsPermission((short) 0777));
        setPermissionsRecursive(fs, outputDir);

        // ── Execute bulk load ──
        System.out.println("  Loading HFiles into table...");

        // HBase 2.x API
        BulkLoadHFilesTool bulkLoadTool = new BulkLoadHFilesTool(conf);
        int loadResult = bulkLoadTool.run(new String[] {
            hfileOutputPath,
            SensorTableManager.TABLE_NAME.getNameAsString()
        });

        long phase2Time = (System.currentTimeMillis() - phase2Start) / 1000;

        if (loadResult == 0) {
            System.out.println("\n  ✅ Phase 2 COMPLETE");
            System.out.printf("    Load time: %d seconds%n", phase2Time);
        } else {
            System.err.println("\n  🔴 Phase 2 FAILED (exit code: " + loadResult + ")");
            return 1;
        }

        // ══════════════════════════════════════════
        // PHASE 3: Validation
        // ══════════════════════════════════════════

        System.out.println("\n  ┌──────────────────────────────────────┐");
        System.out.println("  │  PHASE 3: VALIDATION                   │");
        System.out.println("  └──────────────────────────────────────┘\n");

        try (Connection connection = ConnectionFactory.createConnection(conf);
             Table table = connection.getTable(SensorTableManager.TABLE_NAME)) {

            // Quick row count (sample first and last regions)
            System.out.println("  Sampling table for validation...");

            Scan countScan = new Scan()
                .setFilter(new org.apache.hadoop.hbase.filter.FirstKeyOnlyFilter())
                .setCaching(10000);

            long rowCount = 0;
            try (ResultScanner scanner = table.getScanner(countScan)) {
                for (Result r : scanner) {
                    rowCount++;
                    if (rowCount % 1000000 == 0) {
                        System.out.printf("    Counted %,d rows...%n", rowCount);
                    }
                }
            }

            System.out.printf("    Total rows in table: %,d%n", rowCount);
        }

        // ── Final summary ──
        long totalTime = (System.currentTimeMillis() - phase1Start) / 1000;

        System.out.println("\n═══════════════════════════════════════════════════════");
        System.out.println("  BULK LOAD COMPLETE — FINAL REPORT");
        System.out.println("═══════════════════════════════════════════════════════");
        System.out.printf("  Phase 1 (HFile generation): %d seconds%n", 
            (System.currentTimeMillis() - phase1Start) / 1000 - phase2Time);
        System.out.printf("  Phase 2 (Load into HBase):  %d seconds%n", phase2Time);
        System.out.printf("  Total pipeline time:        %d seconds (%.1f minutes)%n",
            totalTime, totalTime / 60.0);
        System.out.println("  Live query impact:          ZERO (bulk load path)");
        System.out.println("═══════════════════════════════════════════════════════");

        // ── Clean up temp HFiles ──
        System.out.println("\n  Cleaning up temporary HFiles...");
        fs.delete(outputDir, true);
        System.out.println("  ✅ Cleanup complete");

        return 0;
    }

    /**
     * Recursively set permissions on HFile directory.
     * Required so HBase RegionServers can read/move the files.
     */
    private void setPermissionsRecursive(FileSystem fs, Path path) throws IOException {
        org.apache.hadoop.fs.FileStatus[] statuses = fs.listStatus(path);
        if (statuses != null) {
            for (org.apache.hadoop.fs.FileStatus status : statuses) {
                fs.setPermission(status.getPath(),
                    new org.apache.hadoop.fs.permission.FsPermission((short) 0777));
                if (status.isDirectory()) {
                    setPermissionsRecursive(fs, status.getPath());
                }
            }
        }
    }

    public static void main(String[] args) throws Exception {
        Configuration conf = HBaseConfiguration.create();
        int exitCode = ToolRunner.run(conf, new BulkLoadDriver(), args);
        System.exit(exitCode);
    }
}

    public static void generateTextData(
            String outputPath, int numSensors, 
            int readingsPerSensor, long startTimestamp) throws IOException {

        Random rand = new Random(42);

                new FileWriter(outputPath), 1024 * 1024)) {

                String type = types[s % types.length];

                float baseHumidity = 30.0f + rand.nextFloat() * 40.0f;

                                 rand.nextFloat() < 0.80 ? 1 : 2;

                        sensorId, timestamp, temp, humidity, pressure,
                        airQuality, noiseLevel, status));
                    writer.newLine();

                    double elapsed = (System.currentTimeMillis() - writeStart) / 1000.0;

                    System.out.printf("    %d/%d sensors (%.0f%%) — %.0f seconds%n",
                        s + 1, numSensors, pct, elapsed);
                }
            }
        }

        File f = new File(outputPath);

package com.smartcity.bulkload;

import com.iot.sensor.SensorReading;
import com.smartcity.hbase.SensorRowKey;
import com.smartcity.hbase.SensorTableManager;

import org.apache.hadoop.hbase.KeyValue;
import org.apache.hadoop.hbase.io.ImmutableBytesWritable;
import org.apache.hadoop.hbase.util.Bytes;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.mapreduce.Mapper;

import java.io.IOException;
import java.util.TreeMap;

/**
 * Reads binary sensor data via SensorBinaryInputFormat (Project 32)
 * and emits sorted KeyValues for HFile generation.
 *
 * Input types: <LongWritable, SensorReading> from SensorRecordReader
 * Output types: <ImmutableBytesWritable, KeyValue> for HFileOutputFormat2
 *
 * This bridges THREE projects:
 * → Project 32: Custom InputFormat reads proprietary binary
 * → Project 35: SensorRowKey builds optimized row keys
 * → Project 36: HFile bulk load bypasses write path
 */
public class BulkLoadBinaryMapper
        extends Mapper<LongWritable, SensorReading, ImmutableBytesWritable, KeyValue> {

    private ImmutableBytesWritable outputKey = new ImmutableBytesWritable();

    enum BinaryBulkCounters {
        RECORDS_PROCESSED,
        KEYVALUES_EMITTED,
        RECORDS_FILTERED_OFFLINE,
        RECORDS_FILTERED_INVALID_TEMP
    }

    @Override
    protected void map(LongWritable byteOffset, SensorReading reading, Context context)
            throws IOException, InterruptedException {

        context.getCounter(BinaryBulkCounters.RECORDS_PROCESSED).increment(1);

        // ── Filter offline sensors (no useful data) ──
        if (reading.getStatus() == SensorReading.STATUS_OFFLINE) {
            context.getCounter(BinaryBulkCounters.RECORDS_FILTERED_OFFLINE).increment(1);
            return;
        }

        // ── Validate temperature range ──
        if (reading.getTemperature() < -50.0f || reading.getTemperature() > 150.0f) {
            context.getCounter(BinaryBulkCounters.RECORDS_FILTERED_INVALID_TEMP).increment(1);
            return;
        }

        // ── Build row key (Project 35 design) ──
        byte[] rowKey = SensorRowKey.createRowKey(
            reading.getSensorId(), reading.getTimestamp());
        outputKey.set(rowKey);

        long timestamp = reading.getTimestamp();

        // ── Build sorted column map ──
        // CRITICAL: columns MUST be in alphabetical order for HFile
        // TreeMap with Bytes.BYTES_COMPARATOR guarantees this
        TreeMap<byte[], byte[]> columns = new TreeMap<>(Bytes.BYTES_COMPARATOR);

        columns.put(SensorTableManager.COL_HUMIDITY,
                    Bytes.toBytes(reading.getHumidity()));
        columns.put(SensorTableManager.COL_PRESSURE,
                    Bytes.toBytes(reading.getPressure()));
        columns.put(SensorTableManager.COL_SENSOR_ID,
                    Bytes.toBytes(reading.getSensorId()));
        columns.put(SensorTableManager.COL_STATUS,
                    new byte[]{(byte) reading.getStatus()});
        columns.put(SensorTableManager.COL_TEMPERATURE,
                    Bytes.toBytes(reading.getTemperature()));

        // ── Emit sorted KeyValues ──
        for (java.util.Map.Entry<byte[], byte[]> entry : columns.entrySet()) {
            KeyValue kv = new KeyValue(
                rowKey,
                SensorTableManager.CF_DATA,
                entry.getKey(),
                timestamp,
                entry.getValue()
            );
            context.write(outputKey, kv);
            context.getCounter(BinaryBulkCounters.KEYVALUES_EMITTED).increment(1);
        }
    }
}

package com.smartcity.bulkload;

import com.iot.sensor.SensorBinaryInputFormat;
import com.smartcity.hbase.SensorTableManager;

import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.conf.Configured;
import org.apache.hadoop.fs.FileSystem;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.hbase.HBaseConfiguration;
import org.apache.hadoop.hbase.KeyValue;
import org.apache.hadoop.hbase.client.*;
import org.apache.hadoop.hbase.io.ImmutableBytesWritable;
import org.apache.hadoop.hbase.mapreduce.HFileOutputFormat2;
import org.apache.hadoop.hbase.tool.BulkLoadHFilesTool;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;
import org.apache.hadoop.util.Tool;
import org.apache.hadoop.util.ToolRunner;

public class BulkLoadBinaryDriver extends Configured implements Tool {

    @Override
    public int run(String[] args) throws Exception {

        if (args.length < 2) {
            System.err.println("Usage: BulkLoadBinaryDriver <binary-input-path> <temp-hfile-path>");
            return 1;
        }

        String inputPath = args[0];
        String hfileOutputPath = args[1];

        Configuration conf = HBaseConfiguration.create(getConf());

        // Configure byte order for binary reading (Project 32)
        conf.set("sensor.byte.order", "BIG_ENDIAN");

        FileSystem fs = FileSystem.get(conf);
        Path outputDir = new Path(hfileOutputPath);
        fs.delete(outputDir, true);

        System.out.println("═══════════════════════════════════════════════════════");
        System.out.println("  BINARY-TO-HBASE BULK LOAD PIPELINE");
        System.out.println("  (Project 32 InputFormat → Project 35 RowKey → HFiles)");
        System.out.println("═══════════════════════════════════════════════════════\n");

        // ══════════════════════════════════════════
        // PHASE 1: Binary files → HFiles
        // ══════════════════════════════════════════

        System.out.println("  PHASE 1: Generating HFiles from binary sensor data...\n");
        long phase1Start = System.currentTimeMillis();

        try (Connection connection = ConnectionFactory.createConnection(conf)) {
            Table table = connection.getTable(SensorTableManager.TABLE_NAME);
            RegionLocator locator = connection.getRegionLocator(SensorTableManager.TABLE_NAME);

            Job job = Job.getInstance(conf, "Binary Sensor → HBase Bulk Load");
            job.setJarByClass(BulkLoadBinaryDriver.class);

            // ── Use Project 32's custom InputFormat for binary files ──
            job.setInputFormatClass(SensorBinaryInputFormat.class);
            FileInputFormat.addInputPath(job, new Path(inputPath));

            // ── Use our binary-aware mapper ──
            job.setMapperClass(BulkLoadBinaryMapper.class);
            job.setMapOutputKeyClass(ImmutableBytesWritable.class);
            job.setMapOutputValueClass(KeyValue.class);

            FileOutputFormat.setOutputPath(job, outputDir);

            // ── Configure HFile generation (same magic call) ──
            HFileOutputFormat2.configureIncrementalLoad(job, table, locator);

            System.out.printf("    Regions: %d%n", locator.getAllRegionLocations().size());
            System.out.printf("    Reducers: %d%n", job.getNumReduceTasks());
            System.out.println("    Starting MapReduce job...\n");

            boolean success = job.waitForCompletion(true);

            if (!success) {
                System.err.println("  🔴 HFile generation FAILED");
                return 1;
            }

            long phase1Time = (System.currentTimeMillis() - phase1Start) / 1000;

            long processed = job.getCounters()
                .findCounter(BulkLoadBinaryMapper.BinaryBulkCounters.RECORDS_PROCESSED).getValue();
            long kvEmitted = job.getCounters()
                .findCounter(BulkLoadBinaryMapper.BinaryBulkCounters.KEYVALUES_EMITTED).getValue();
            long filtered = job.getCounters()
                .findCounter(BulkLoadBinaryMapper.BinaryBulkCounters.RECORDS_FILTERED_OFFLINE).getValue()
                + job.getCounters()
                .findCounter(BulkLoadBinaryMapper.BinaryBulkCounters.RECORDS_FILTERED_INVALID_TEMP).getValue();

            System.out.printf("\n  ✅ Phase 1 complete in %d seconds%n", phase1Time);
            System.out.printf("    Records processed:  %,d%n", processed);
            System.out.printf("    Records filtered:   %,d%n", filtered);
            System.out.printf("    KeyValues emitted:  %,d%n", kvEmitted);

            table.close();
            locator.close();
        }

        // ══════════════════════════════════════════
        // PHASE 2: Load HFiles → HBase
        // ══════════════════════════════════════════

        System.out.println("\n  PHASE 2: Loading HFiles into HBase...");
        long phase2Start = System.currentTimeMillis();

        // Set permissions for HBase access
        setPermissionsRecursive(fs, outputDir);

        BulkLoadHFilesTool bulkLoadTool = new BulkLoadHFilesTool(conf);
        int loadResult = bulkLoadTool.run(new String[]{
            hfileOutputPath,
            SensorTableManager.TABLE_NAME.getNameAsString()
        });

        long phase2Time = (System.currentTimeMillis() - phase2Start) / 1000;

        if (loadResult == 0) {
            System.out.printf("  ✅ Phase 2 complete in %d seconds%n", phase2Time);
        } else {
            System.err.println("  🔴 Phase 2 FAILED");
            return 1;
        }

        // Cleanup
        fs.delete(outputDir, true);

        long totalTime = (System.currentTimeMillis() - phase1Start) / 1000;
        System.out.println("\n═══════════════════════════════════════════════════════");
        System.out.printf("  TOTAL PIPELINE TIME: %d seconds (%.1f minutes)%n",
            totalTime, totalTime / 60.0);
        System.out.println("  Live query impact: ZERO");
        System.out.println("═══════════════════════════════════════════════════════");

        return 0;
    }

    private void setPermissionsRecursive(FileSystem fs, Path path) throws java.io.IOException {
        org.apache.hadoop.fs.FileStatus[] statuses = fs.listStatus(path);
        if (statuses != null) {
            for (org.apache.hadoop.fs.FileStatus status : statuses) {
                fs.setPermission(status.getPath(),
                    new org.apache.hadoop.fs.permission.FsPermission((short) 0777));
                if (status.isDirectory()) {
                    setPermissionsRecursive(fs, status.getPath());
                }
            }
        }
    }

    public static void main(String[] args) throws Exception {
        int exitCode = ToolRunner.run(
            HBaseConfiguration.create(), new BulkLoadBinaryDriver(), args);
        System.exit(exitCode);
    }
}

package com.smartcity.bulkload;

import com.smartcity.hbase.SensorRowKey;
import com.smartcity.hbase.SensorTableManager;

import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.hbase.HBaseConfiguration;
import org.apache.hadoop.hbase.client.*;
import org.apache.hadoop.hbase.filter.FirstKeyOnlyFilter;
import org.apache.hadoop.hbase.util.Bytes;

import java.io.IOException;
import java.util.*;

public class BulkLoadValidator {

    /**
     * Comprehensive post-load validation.
     */
    public static void validate(Configuration conf, long expectedRecords) throws IOException {

        System.out.println("═══════════════════════════════════════════════════════");
        System.out.println("  BULK LOAD VALIDATION");
        System.out.println("═══════════════════════════════════════════════════════\n");

        try (Connection connection = ConnectionFactory.createConnection(conf);
             Table table = connection.getTable(SensorTableManager.TABLE_NAME)) {

            boolean allPassed = true;

            // ── Check 1: Row count ──
            System.out.println("  [1/5] Counting rows...");
            long rowCount = countRows(table);
            boolean countMatch = (expectedRecords <= 0) || 
                                 (Math.abs(rowCount - expectedRecords) < expectedRecords * 0.01);
            System.out.printf("    Rows found: %,d%n", rowCount);
            if (expectedRecords > 0) {
                System.out.printf("    Expected:   %,d%n", expectedRecords);
                System.out.printf("    Match:      %s%n", countMatch ? "✅ YES (within 1%)" : "🔴 NO");
                if (!countMatch) allPassed = false;
            }

            // ── Check 2: Salt distribution ──
            System.out.println("\n  [2/5] Checking salt distribution...");
            Map<Integer, Long> saltDist = checkSaltDistribution(table);
            long minPerSalt = Long.MAX_VALUE;
            long maxPerSalt = 0;
            long totalFromDist = 0;

            for (Map.Entry<Integer, Long> entry : saltDist.entrySet()) {
                minPerSalt = Math.min(minPerSalt, entry.getValue());
                maxPerSalt = Math.max(maxPerSalt, entry.getValue());
                totalFromDist += entry.getValue();
            }

            double avgPerSalt = totalFromDist / (double) Math.max(1, saltDist.size());
            double skewRatio = maxPerSalt / Math.max(1.0, avgPerSalt);

            System.out.printf("    Active salt buckets: %d%n", saltDist.size());
            System.out.printf("    Min/Avg/Max per bucket: %,d / %.0f / %,d%n",
                minPerSalt, avgPerSalt, maxPerSalt);
            System.out.printf("    Skew ratio: %.2fx %s%n", skewRatio,
                skewRatio < 3.0 ? "✅ BALANCED" : "⚠️ SKEWED");
            if (skewRatio >= 3.0) allPassed = false;

            // ── Check 3: Data readability (sample reads) ──
            System.out.println("\n  [3/5] Sampling data for readability...");
            int sampleSize = 10;
            int readableCount = 0;

            Scan sampleScan = new Scan()
                .addFamily(SensorTableManager.CF_DATA)
                .setLimit(sampleSize)
                .setCaching(sampleSize);

            try (ResultScanner scanner = table.getScanner(sampleScan)) {
                for (Result row : scanner) {
                    if (row != null && !row.isEmpty()) {
                        byte[] sid = row.getValue(SensorTableManager.CF_DATA,
                                                  SensorTableManager.COL_SENSOR_ID);
                        byte[] temp = row.getValue(SensorTableManager.CF_DATA,
                                                   SensorTableManager.COL_TEMPERATURE);

                        if (sid != null && temp != null) {
                            SensorRowKey key = SensorRowKey.parse(row.getRow());
                            String sensorId = Bytes.toString(sid);
                            float temperature = Bytes.toFloat(temp);

                            System.out.printf("    Row: %s | %s | %.1f°C%n",
                                sensorId, key.getTimestampString(), temperature);
                            readableCount++;
                        }
                    }
                }
            }

            boolean readable = readableCount == sampleSize;
            System.out.printf("    Readable: %d/%d %s%n", readableCount, sampleSize,
                readable ? "✅ ALL OK" : "🔴 SOME FAILED");
            if (!readable) allPassed = false;

            // ── Check 4: Row key ordering ──
            System.out.println("\n  [4/5] Verifying row key ordering...");
            boolean ordered = verifyOrdering(table, 1000);
            System.out.printf("    Ordering: %s%n", ordered ? "✅ CORRECT" : "🔴 VIOLATED");
            if (!ordered) allPassed = false;

            // ── Check 5: Reverse timestamp correctness ──
            System.out.println("\n  [5/5] Verifying reverse timestamp (newest first)...");
            boolean reverseCorrect = verifyReverseTimestamp(table);
            System.out.printf("    Newest first: %s%n", 
                reverseCorrect ? "✅ CORRECT" : "🔴 WRONG ORDER");
            if (!reverseCorrect) allPassed = false;

            // ── Final verdict ──
            System.out.println("\n═══════════════════════════════════════════════════════");
            if (allPassed) {
                System.out.println("  ✅ ALL VALIDATION CHECKS PASSED");
            } else {
                System.out.println("  🔴 SOME VALIDATION CHECKS FAILED — INVESTIGATE");
            }
            System.out.println("═══════════════════════════════════════════════════════");
        }
    }

    /**
     * Count total rows using FirstKeyOnlyFilter for speed.
     */
    private static long countRows(Table table) throws IOException {
        Scan scan = new Scan()
            .setFilter(new FirstKeyOnlyFilter())
            .setCaching(10000);

        long count = 0;
        try (ResultScanner scanner = table.getScanner(scan)) {
            for (Result r : scanner) {
                count++;
                if (count % 1000000 == 0) {
                    System.out.printf("      Counted %,d rows...%n", count);
                }
            }
        }
        return count;
    }

    /**
     * Count rows per salt bucket to verify distribution.
     */
    private static Map<Integer, Long> checkSaltDistribution(Table table) throws IOException {
        Map<Integer, Long> distribution = new TreeMap<>();

        Scan scan = new Scan()
            .setFilter(new FirstKeyOnlyFilter())
            .setCaching(10000);

        try (ResultScanner scanner = table.getScanner(scan)) {
            for (Result row : scanner) {
                int salt = row.getRow()[0] & 0xFF;
                distribution.merge(salt, 1L, Long::sum);
            }
        }

        return distribution;
    }

    /**
     * Verify row keys are in lexicographic order.
     */
    private static boolean verifyOrdering(Table table, int sampleSize) throws IOException {
        Scan scan = new Scan()
            .setFilter(new FirstKeyOnlyFilter())
            .setLimit(sampleSize)
            .setCaching(sampleSize);

        byte[] previousKey = null;
        boolean ordered = true;

        try (ResultScanner scanner = table.getScanner(scan)) {
            for (Result row : scanner) {
                byte[] currentKey = row.getRow();
                if (previousKey != null) {
                    if (Bytes.compareTo(previousKey, currentKey) >= 0) {
                        ordered = false;
                        break;
                    }
                }
                previousKey = currentKey;
            }
        }

        return ordered;
    }

    /**
     * Verify that for a single sensor, newer readings come first.
     */
    private static boolean verifyReverseTimestamp(Table table) throws IOException {
        // Get the first row to find a sensor ID
        Scan firstScan = new Scan()
            .addFamily(SensorTableManager.CF_DATA)
            .setLimit(1);

        String testSensorId = null;
        try (ResultScanner scanner = table.getScanner(firstScan)) {
            Result first = scanner.next();
            if (first != null) {
                byte[] sid = first.getValue(SensorTableManager.CF_DATA,
                                            SensorTableManager.COL_SENSOR_ID);
                if (sid != null) {
                    testSensorId = Bytes.toString(sid);
                }
            }
        }

        if (testSensorId == null) return false;

        // Scan first 10 readings for this sensor
        byte[] startRow = SensorRowKey.createScanStartRow(testSensorId);
        byte[] stopRow = SensorRowKey.createScanStopRow(testSensorId);

        Scan sensorScan = new Scan()
            .withStartRow(startRow)
            .withStopRow(stopRow)
            .addFamily(SensorTableManager.CF_DATA)
            .setLimit(10)
            .setCaching(10);

        long previousTimestamp = Long.MAX_VALUE;
        boolean correctOrder = true;

        try (ResultScanner scanner = table.getScanner(sensorScan)) {
            for (Result row : scanner) {
                SensorRowKey key = SensorRowKey.parse(row.getRow());
                long ts = key.getOriginalTimestamp();

                if (ts > previousTimestamp) {
                    // Newer timestamp came AFTER older → wrong order
                    correctOrder = false;
                    break;
                }
                previousTimestamp = ts;
            }
        }

        System.out.printf("    Test sensor: %s — timestamps decrease: %s%n",
            testSensorId, correctOrder ? "YES (newest first)" : "NO");

        return correctOrder;
    }

    public static void main(String[] args) throws Exception {
        Configuration conf = HBaseConfiguration.create();
        if (args.length > 0) {
            conf.set("hbase.zookeeper.quorum", args[0]);
        }

        long expectedRecords = args.length > 1 ? Long.parseLong(args[1]) : -1;
        validate(conf, expectedRecords);
    }
}
