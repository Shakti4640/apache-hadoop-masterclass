✅ WHAT SHOULD DO

1. MAKE RAW DATA IMMUTABLE
   → NEVER modify files in /data/raw/
   → Raw data is the "source of truth" — always reprocessable
   → Cleaning creates NEW files in /data/cleaned/
   → If cleaning logic changes → reprocess from raw

2. USE DATE PARTITIONING EVERYWHERE
   → Every directory includes dt=YYYY-MM-DD
   → Enables partition pruning in Hive (Project 25)
   → Enables easy data lifecycle management (delete old partitions)

3. BUILD IDEMPOTENT STAGES
   → Every stage can be re-run safely
   → Use INSERT OVERWRITE (not INSERT INTO) in Hive
   → Use HDFS -rm -r before writing cleaned data
   → If pipeline fails at stage 3 → restart from stage 3 (not stage 1)

4. IMPLEMENT QUALITY GATES
   → After cleaning: check that malformed record % is below threshold
   → If > 5% malformed → STOP pipeline → alert → investigate
   → Do NOT load garbage into the warehouse

5. SEPARATE COMPUTE FROM STORAGE
   → Raw zone: write-optimized format (text, Avro)
   → Warehouse zone: read-optimized format (ORC)
   → Serving zone: lookup-optimized (HBase)
   → Each zone serves a different access pattern

6. VERSION EVERYTHING
   → Pipeline code in Git
   → Oozie workflow definitions in Git
   → Hive DDL statements in Git
   → Configuration files in Git
   → Test data and test scripts in Git (Project 38)

🚫 WHAT SHOULD NOT DO

1. DO NOT SKIP THE CLEANING STAGE
   → Raw data is ALWAYS dirty — no exceptions
   → Loading raw data directly into warehouse → corrupt analytics
   → "We'll clean it later" → you never will

2. DO NOT USE INSERT INTO FOR DAILY LOADS
   → INSERT INTO appends — running pipeline twice = duplicate data
   → INSERT OVERWRITE replaces — idempotent, safe to re-run
   → Always prefer OVERWRITE for batch pipeline stages

3. DO NOT HARDCODE DATES IN PIPELINE
   → Oozie passes ${coord:formatTime(coord:nominalTime(), 'yyyy-MM-dd')}
   → Pipeline must work for ANY date, not just "today"
   → Enables backfilling historical data

4. DO NOT RUN ALL STAGES IN SEQUENCE IF INDEPENDENT
   → Sqoop for transactions and Sqoop for products are INDEPENDENT
   → Run them in PARALLEL using Oozie fork/join (Project 31)
   → Web log cleaning and transaction cleaning are INDEPENDENT
   → Parallelize wherever possible — reduces total pipeline time

5. DO NOT IGNORE FAILURE HANDLING
   → Every Oozie action needs an error-to transition
   → Failed Sqoop import → do NOT proceed to cleaning
   → Failed cleaning → do NOT proceed to transformation
   → Pipeline must FAIL FAST and LOUDLY

6. DO NOT LOAD HBASE DIRECTLY WITH PUT OPERATIONS
   → 80 million rows via individual puts = HOURS + RegionServer stress
   → Use bulk load (Project 36): generate HFiles → atomic load
   → Orders of magnitude faster and safer

7. DO NOT MIX ETL AND ANALYTICS IN THE SAME YARN QUEUE
   → ETL runs at night — needs maximum throughput
   → Analytics runs during the day — needs fast response
   → Use separate queues from Project 39

🚫 WHEN NOT TO USE

1. DO NOT USE THIS BATCH PIPELINE FOR REAL-TIME DASHBOARDS
   → Pipeline runs nightly — data is always T-1
   → For real-time: Kafka + Flink/Spark Streaming
   → (Bridge to Project 47: Lambda Architecture)

2. DO NOT USE MAPREDUCE FOR SIMPLE TRANSFORMATIONS
   → If transformation is pure SQL → use Hive directly
   → MapReduce cleaning is for complex validation logic
   → If cleaning is simple column filtering → Hive WHERE clause suffices

3. DO NOT USE THIS ARCHITECTURE FOR < 10GB/DAY
   → Overhead of Flume + Sqoop + MapReduce + Hive + Oozie
   → For small data: simple Python script + PostgreSQL
   → Hadoop shines at 100GB+ daily volumes

4. DO NOT USE OOZIE FOR SUB-MINUTE SCHEDULING
   → Oozie coordinator minimum frequency: 5 minutes
   → For sub-minute: use external scheduler (Airflow, Luigi)
   → Or streaming architecture (Kafka + Flink)

5. DO NOT BUILD MONOLITHIC PIPELINE
   → Each stage should be independently testable
   → Each stage should be independently re-runnable
   → Tight coupling between stages = fragile pipeline

 ─── 12:30 AM — STAGE 2a: CLEAN WEB LOGS ─────────────

 Oozie → MapReduce Action (map-only, Project 11):

   INPUT:  /data/raw/weblogs/2024/01/14/*/* (all hours)
   OUTPUT: /data/cleaned/weblogs/dt=2024-01-14/

   Each mapper:
   → Reads raw log line:
     192.168.1.1 - - [14/Jan/2024:13:45:22 +0000] "GET /product/shoe-123 HTTP/1.1" 200 4523
   → Validates: Is HTTP method valid? Is status code numeric? Is URL parseable?

   → Writes compressed output (GzipCodec, Project 14)

   → Ratio: 0.0075% → BELOW 5% threshold → PASS

   PARTITION (dt='2024-01-14', hr)

package com.ecommerce.pipeline;

import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.NullWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Mapper;

import java.io.IOException;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class WebLogCleanerMapper
        extends Mapper<LongWritable, Text, Text, NullWritable> {

    // Counters for data quality tracking (Project 7)
    public enum LogCounters {
        TOTAL_LINES,
        VALID_LINES,
        MALFORMED_LINES,
        MISSING_PRODUCT_ID,
        INVALID_STATUS_CODE,
        BOT_TRAFFIC_FILTERED,
        EMPTY_LINES
    }

    // Apache/Nginx combined log format regex
    // Example: 192.168.1.1 - - [14/Jan/2024:13:45:22 +0000]
    //          "GET /product/shoe-123 HTTP/1.1" 200 4523
    //          "http://example.com" "Mozilla/5.0..."
    private static final Pattern LOG_PATTERN = Pattern.compile(
        "^(\\S+)"                          // IP address
        + " \\S+ \\S+"                     // ident, authuser
        + " \\[([^]]+)]"                   // timestamp
        + " \"(\\S+)"                      // HTTP method
        + " (\\S+)"                        // request URL
        + " \\S+\""                        // HTTP version
        + " (\\d{3})"                      // status code
        + " (\\d+|-)"                      // response size
        + "(?:"                            // optional fields
        + " \"([^\"]*)\""                  // referer
        + " \"([^\"]*)\""                  // user agent
        + ")?"
    );

    // Known bot user-agent patterns to filter
    private static final String[] BOT_PATTERNS = {
        "Googlebot", "bingbot", "Baiduspider",
        "YandexBot", "DotBot", "SemrushBot"
    };

    // Product URL pattern: /product/PRODUCT-ID
    private static final Pattern PRODUCT_URL_PATTERN =
        Pattern.compile("^/product/([\\w-]+)");

    private final Text outputValue = new Text();

    @Override
    protected void map(LongWritable key, Text value, Context context)
            throws IOException, InterruptedException {

        String line = value.toString().trim();

        // Skip empty lines
        if (line.isEmpty()) {
            context.getCounter(LogCounters.EMPTY_LINES).increment(1);
            return;
        }

        context.getCounter(LogCounters.TOTAL_LINES).increment(1);

        // Parse log line with regex
        Matcher matcher = LOG_PATTERN.matcher(line);
        if (!matcher.find()) {
            context.getCounter(LogCounters.MALFORMED_LINES).increment(1);
            return;
        }

        // Extract fields
        String userIp = matcher.group(1);
        String timestamp = matcher.group(2);
        String httpMethod = matcher.group(3);
        String requestUrl = matcher.group(4);
        String statusCodeStr = matcher.group(5);
        String responseSizeStr = matcher.group(6);
        String referer = matcher.groupCount() >= 7 ? matcher.group(7) : "-";
        String userAgent = matcher.groupCount() >= 8 ? matcher.group(8) : "-";

        // Validate status code
        int statusCode;
        try {
            statusCode = Integer.parseInt(statusCodeStr);
            if (statusCode < 100 || statusCode > 599) {
                context.getCounter(LogCounters.INVALID_STATUS_CODE).increment(1);
                context.getCounter(LogCounters.MALFORMED_LINES).increment(1);
                return;
            }
        } catch (NumberFormatException e) {
            context.getCounter(LogCounters.INVALID_STATUS_CODE).increment(1);
            context.getCounter(LogCounters.MALFORMED_LINES).increment(1);
            return;
        }

        // Filter bot traffic
        if (userAgent != null) {
            for (String bot : BOT_PATTERNS) {
                if (userAgent.contains(bot)) {
                    context.getCounter(LogCounters.BOT_TRAFFIC_FILTERED).increment(1);
                    return;
                }
            }
        }

        // Extract product ID from URL
        String productId = "-";
        Matcher productMatcher = PRODUCT_URL_PATTERN.matcher(requestUrl);
        if (productMatcher.find()) {
            productId = productMatcher.group(1);
        } else {
            context.getCounter(LogCounters.MISSING_PRODUCT_ID).increment(1);
            // Still emit — not all URLs are product pages
        }

        // Parse response size
        long responseSize = 0;
        if (!"-".equals(responseSizeStr)) {
            try {
                responseSize = Long.parseLong(responseSizeStr);
            } catch (NumberFormatException e) {
                responseSize = 0;
            }
        }

        // Emit cleaned tab-separated record
        // Matches cleaned_zone.weblogs table schema
        String cleanedRecord = String.join("\t",
            timestamp,
            userIp,
            httpMethod,
            requestUrl,
            productId,
            String.valueOf(statusCode),
            String.valueOf(responseSize),
            userAgent != null ? userAgent : "-",
            referer != null ? referer : "-"
        );

        outputValue.set(cleanedRecord);
        context.write(outputValue, NullWritable.get());
        context.getCounter(LogCounters.VALID_LINES).increment(1);
    }
}

import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;
import org.apache.hadoop.util.Tool;
import org.apache.hadoop.util.ToolRunner;

public class WebLogCleanerDriver extends Configured implements Tool {

    @Override
    public int run(String[] args) throws Exception {
        if (args.length != 2) {
            System.err.println("Usage: WebLogCleanerDriver <input> <output>");
            return -1;
        }

        Configuration conf = getConf();
        Job job = Job.getInstance(conf, "WebLog Cleaner - " + args[0]);
        job.setJarByClass(WebLogCleanerDriver.class);

        // Map-only job — zero reducers (Project 11 pattern)
        job.setMapperClass(WebLogCleanerMapper.class);
        job.setNumReduceTasks(0);

        // Output types
        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(NullWritable.class);

        // Compress output (Project 14)
        FileOutputFormat.setCompressOutput(job, true);
        FileOutputFormat.setOutputCompressorClass(job, GzipCodec.class);

        // Input/Output paths
        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        // Submit to ETL queue (Project 39)
        job.getConfiguration().set("mapreduce.job.queuename", "root.production.etl");

        boolean success = job.waitForCompletion(true);

        if (success) {
            // Print quality metrics from counters (Project 7)
            long totalLines = job.getCounters()
                .findCounter(WebLogCleanerMapper.LogCounters.TOTAL_LINES).getValue();
            long validLines = job.getCounters()
                .findCounter(WebLogCleanerMapper.LogCounters.VALID_LINES).getValue();
            long malformedLines = job.getCounters()
                .findCounter(WebLogCleanerMapper.LogCounters.MALFORMED_LINES).getValue();
            long botFiltered = job.getCounters()
                .findCounter(WebLogCleanerMapper.LogCounters.BOT_TRAFFIC_FILTERED).getValue();

            double malformedRate = totalLines > 0
                ? (double) malformedLines / totalLines * 100 : 0;

            System.out.println("=== WEB LOG CLEANING REPORT ===");
            System.out.println("Total Lines:     " + totalLines);
            System.out.println("Valid Lines:     " + validLines);
            System.out.println("Malformed Lines: " + malformedLines);
            System.out.println("Bot Filtered:    " + botFiltered);
            System.out.printf("Malformed Rate:  %.2f%%\n", malformedRate);

            // QUALITY GATE: Fail if malformed rate exceeds threshold
            if (malformedRate > 5.0) {
                System.err.println("QUALITY GATE FAILED: Malformed rate " +
                    malformedRate + "% exceeds 5% threshold");
                return 2; // Special exit code for quality gate failure
            }
        }

        return success ? 0 : 1;
    }

    public static void main(String[] args) throws Exception {
        int exitCode = ToolRunner.run(new Configuration(), new WebLogCleanerDriver(), args);
        System.exit(exitCode);
    }
}

package com.ecommerce.pipeline;

import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.NullWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Mapper;

import java.io.IOException;

public class TransactionCleanerMapper
        extends Mapper<LongWritable, Text, Text, NullWritable> {

    public enum TxnCounters {
        TOTAL_RECORDS,
        VALID_RECORDS,
        NULL_ORDER_ID,
        NULL_CUSTOMER_ID,
        INVALID_AMOUNT,
        NEGATIVE_QUANTITY,
        MALFORMED_RECORDS
    }

    private final Text outputValue = new Text();

    @Override
    protected void map(LongWritable key, Text value, Context context)
            throws IOException, InterruptedException {

        String line = value.toString().trim();
        if (line.isEmpty()) return;

        context.getCounter(TxnCounters.TOTAL_RECORDS).increment(1);

        // Avro deserialized as tab-separated by TextInputFormat over Avro
        // Fields: order_id, customer_id, product_id, quantity, amount,
        //         order_timestamp, payment_method, shipping_address
        String[] fields = line.split("\t");

        if (fields.length < 7) {
            context.getCounter(TxnCounters.MALFORMED_RECORDS).increment(1);
            return;
        }

        // Validate order_id
        String orderIdStr = fields[0].trim();
        if (orderIdStr.isEmpty() || "null".equalsIgnoreCase(orderIdStr)) {
            context.getCounter(TxnCounters.NULL_ORDER_ID).increment(1);
            context.getCounter(TxnCounters.MALFORMED_RECORDS).increment(1);
            return;
        }

        // Validate customer_id
        String customerIdStr = fields[1].trim();
        if (customerIdStr.isEmpty() || "null".equalsIgnoreCase(customerIdStr)) {
            context.getCounter(TxnCounters.NULL_CUSTOMER_ID).increment(1);
            context.getCounter(TxnCounters.MALFORMED_RECORDS).increment(1);
            return;
        }

        // Validate amount
        double amount;
        try {
            amount = Double.parseDouble(fields[4].trim());
            if (amount <= 0) {
                context.getCounter(TxnCounters.INVALID_AMOUNT).increment(1);
                context.getCounter(TxnCounters.MALFORMED_RECORDS).increment(1);
                return;
            }
        } catch (NumberFormatException e) {
            context.getCounter(TxnCounters.INVALID_AMOUNT).increment(1);
            context.getCounter(TxnCounters.MALFORMED_RECORDS).increment(1);
            return;
        }

        // Validate quantity
        int quantity;
        try {
            quantity = Integer.parseInt(fields[3].trim());
            if (quantity <= 0) {
                context.getCounter(TxnCounters.NEGATIVE_QUANTITY).increment(1);
                context.getCounter(TxnCounters.MALFORMED_RECORDS).increment(1);
                return;
            }
        } catch (NumberFormatException e) {
            context.getCounter(TxnCounters.NEGATIVE_QUANTITY).increment(1);
            context.getCounter(TxnCounters.MALFORMED_RECORDS).increment(1);
            return;
        }

        // Emit cleaned record (strip shipping_address for privacy)
        String cleanedRecord = String.join("\t",
            orderIdStr,
            customerIdStr,
            fields[2].trim(),   // product_id
            String.valueOf(quantity),
            String.valueOf(amount),
            fields[5].trim(),   // order_timestamp
            fields[6].trim()    // payment_method
        );

        outputValue.set(cleanedRecord);
        context.write(outputValue, NullWritable.get());
        context.getCounter(TxnCounters.VALID_RECORDS).increment(1);
    }
}

package com.ecommerce.pipeline;

import org.apache.hadoop.hbase.KeyValue;
import org.apache.hadoop.hbase.io.ImmutableBytesWritable;
import org.apache.hadoop.hbase.util.Bytes;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Mapper;

import java.io.IOException;

public class HBaseCustomerHistoryMapper
        extends Mapper<LongWritable, Text, ImmutableBytesWritable, KeyValue> {

    // Column family
    private static final byte[] CF_INFO = Bytes.toBytes("info");

    // Column qualifiers
    private static final byte[] COL_TOTAL_ORDERS = Bytes.toBytes("total_orders");
    private static final byte[] COL_TOTAL_SPENT = Bytes.toBytes("total_spent");
    private static final byte[] COL_LAST_ORDER = Bytes.toBytes("last_order_date");
    private static final byte[] COL_TOP_CATEGORIES = Bytes.toBytes("top_categories");
    private static final byte[] COL_RECENT_PRODUCTS = Bytes.toBytes("recent_products");

    private final ImmutableBytesWritable rowKeyWritable = new ImmutableBytesWritable();

    @Override
    protected void map(LongWritable key, Text value, Context context)
            throws IOException, InterruptedException {

        String line = value.toString().trim();
        if (line.isEmpty()) return;

        // Fields: customer_id, total_orders, total_spent,
        //         last_order_date, top_categories, recent_products
        String[] fields = line.split("\t");
        if (fields.length < 6) return;

        long customerId;
        try {
            customerId = Long.parseLong(fields[0].trim());
        } catch (NumberFormatException e) {
            return;
        }

        // Row key design (Project 35):
        // Salt (2 digits) + reverse_customer_id
        // Salt prevents hot-spotting on sequential customer IDs
        int salt = (int) (customerId % 100);
        String rowKey = String.format("%02d_%020d", salt, customerId);

        byte[] rowKeyBytes = Bytes.toBytes(rowKey);
        rowKeyWritable.set(rowKeyBytes);

        long timestamp = System.currentTimeMillis();

        // Emit one KeyValue per column
        // HFileOutputFormat2 requires columns in SORTED order within row
        // Sort order: column family → column qualifier (lexicographic)

        context.write(rowKeyWritable,
            new KeyValue(rowKeyBytes, CF_INFO, COL_LAST_ORDER,
                timestamp, Bytes.toBytes(fields[3].trim())));

        context.write(rowKeyWritable,
            new KeyValue(rowKeyBytes, CF_INFO, COL_RECENT_PRODUCTS,
                timestamp, Bytes.toBytes(fields[5].trim())));

        context.write(rowKeyWritable,
            new KeyValue(rowKeyBytes, CF_INFO, COL_TOP_CATEGORIES,
                timestamp, Bytes.toBytes(fields[4].trim())));

        context.write(rowKeyWritable,
            new KeyValue(rowKeyBytes, CF_INFO, COL_TOTAL_ORDERS,
                timestamp, Bytes.toBytes(fields[1].trim())));

        context.write(rowKeyWritable,
            new KeyValue(rowKeyBytes, CF_INFO, COL_TOTAL_SPENT,
                timestamp, Bytes.toBytes(fields[2].trim())));
    }
}

-- TRANSFORM 4: Daily Revenue Aggregation
-- ════════════════════════════════════════

PARTITION (dt='${PROCESS_DATE}')
SELECT
    category,
    COUNT(*) as order_count,
    ROUND(SUM(amount), 2) as total_revenue,
    ROUND(AVG(amount), 2) as avg_order_value,
    COUNT(DISTINCT customer_id) as unique_customers
FROM warehouse.fact_orders
WHERE dt = '${PROCESS_DATE}'

-- ════════════════════════════════════════

-- ════════════════════════════════════════

PARTITION (dt='${PROCESS_DATE}', hr)

    product_id,
    COUNT(*) as pageview_count,

    hr
FROM warehouse.fact_pageviews
WHERE dt = '${PROCESS_DATE}'
GROUP BY product_id, hr;


-- ════════════════════════════════════════
-- TRANSFORM 6: Customer Purchase History (90-day window)
-- This feeds HBase bulk load
-- ════════════════════════════════════════

INSERT OVERWRITE TABLE warehouse.customer_history
PARTITION (dt='${PROCESS_DATE}')
SELECT
    customer_id,
    COUNT(*) as total_orders,
    ROUND(SUM(amount), 2) as total_spent,
    MAX(CAST(order_timestamp AS STRING)) as last_order_date,
    -- Top 3 categories by spend
    CONCAT_WS(',',
        COLLECT_SET(category)
    ) as top_categories,
    -- Last 10 products purchased
    CONCAT_WS(',',
        SORT_ARRAY(
            COLLECT_SET(
                CONCAT(CAST(order_timestamp AS STRING), ':', product_name)
            )
        )
    ) as recent_products
FROM warehouse.fact_orders
WHERE dt >= date_sub('${PROCESS_DATE}', 90)
  AND dt <= '${PROCESS_DATE}'
GROUP BY customer_id;

-- FILE 8: quality_validation.sql
-- PURPOSE: Data quality checks after transformation
-- Returns non-zero row count if any check FAILS
-- Oozie uses this to decide: continue or alert
-- ══════════════════════════════════════════════════════════════

-- Quality Check 1: Record count variance
-- Compare today's pageview count with 7-day average
-- Alert if > 30% deviation

SELECT 'PAGEVIEW_COUNT_CHECK' as check_name,
    CASE
        WHEN ABS(today_count - avg_7d_count) / avg_7d_count > 0.30
        THEN 'FAIL'
        ELSE 'PASS'
    END as status,
    today_count,
    avg_7d_count
FROM (
    SELECT
        (SELECT COUNT(*) FROM warehouse.fact_pageviews
         WHERE dt = '${PROCESS_DATE}') as today_count,
        (SELECT AVG(daily_count) FROM (
            SELECT dt, COUNT(*) as daily_count
            FROM warehouse.fact_pageviews
            WHERE dt >= date_sub('${PROCESS_DATE}', 7)
              AND dt < '${PROCESS_DATE}'
            GROUP BY dt
        ) t) as avg_7d_count
) checks;

-- Quality Check 2: Revenue sanity
-- Daily revenue should be within 3 standard deviations of 30-day average

SELECT 'REVENUE_CHECK' as check_name,
    CASE
        WHEN ABS(today_rev - avg_30d) > 3 * stddev_30d
        THEN 'FAIL'
        ELSE 'PASS'
    END as status,
    today_rev,
    avg_30d,
    stddev_30d
FROM (
    SELECT
        (SELECT SUM(total_revenue) FROM warehouse.agg_daily_revenue
         WHERE dt = '${PROCESS_DATE}') as today_rev,
        (SELECT AVG(daily_rev) FROM (
            SELECT dt, SUM(total_revenue) as daily_rev
            FROM warehouse.agg_daily_revenue
            WHERE dt >= date_sub('${PROCESS_DATE}', 30)
              AND dt < '${PROCESS_DATE}'
            GROUP BY dt
        ) t) as avg_30d,
        (SELECT STDDEV(daily_rev) FROM (
            SELECT dt, SUM(total_revenue) as daily_rev
            FROM warehouse.agg_daily_revenue
            WHERE dt >= date_sub('${PROCESS_DATE}', 30)
              AND dt < '${PROCESS_DATE}'
            GROUP BY dt
        ) t) as stddev_30d
) checks;

-- Quality Check 3: Null rate in fact_orders
SELECT 'NULL_RATE_CHECK' as check_name,
    CASE
        WHEN null_rate > 0.001 THEN 'FAIL'
        ELSE 'PASS'
    END as status,
    null_rate
FROM (
    SELECT
        CAST(SUM(CASE WHEN customer_id IS NULL THEN 1 ELSE 0 END) AS DOUBLE)
        / COUNT(*) as null_rate
    FROM warehouse.fact_orders
    WHERE dt = '${PROCESS_DATE}'
) checks;

-- Quality Check 4: Customer history count matches
SELECT 'CUSTOMER_HISTORY_CHECK' as check_name,
    CASE
        WHEN hive_count = 0 THEN 'FAIL'
        ELSE 'PASS'
    END as status,
    hive_count
FROM (
    SELECT COUNT(*) as hive_count
    FROM warehouse.customer_history
    WHERE dt = '${PROCESS_DATE}'
) checks;

import json
import urllib.request
import datetime
import sys


class PipelineHealthChecker:
    """
    Monitors the end-to-end pipeline health.
    Checks Oozie coordinator, HDFS data freshness,
    Hive table partitions, and HBase availability.
    """

    def __init__(self, config):
        self.oozie_url = config.get("oozie_url", "http://localhost:11000/oozie")
        self.rm_url = config.get("rm_url", "http://localhost:8088")
        self.namenode_url = config.get("namenode_url", "http://localhost:9870")
        self.alerts = []
        self.today = datetime.date.today()
        self.yesterday = self.today - datetime.timedelta(days=1)

    def check_oozie_coordinator(self, coord_id):
        """Check if last coordinator run succeeded."""
        url = (f"{self.oozie_url}/v1/job/{coord_id}"
               f"?show=info&len=5&order=desc")
        try:
            req = urllib.request.Request(url)
            req.add_header("Accept", "application/json")
            response = urllib.request.urlopen(req, timeout=15)
            data = json.loads(response.read().decode())

            actions = data.get("actions", [])
            if not actions:
                self.alerts.append({
                    "severity": "WARNING",
                    "component": "Oozie",
                    "message": "No coordinator actions found"
                })
                return

            latest = actions[0]
            status = latest.get("status", "UNKNOWN")
            nominal_time = latest.get("nominalTime", "")

            if status == "SUCCEEDED":
                print(f"  ✅ Last pipeline run: SUCCEEDED ({nominal_time})")
            elif status == "RUNNING":
                print(f"  🔄 Pipeline currently RUNNING ({nominal_time})")
            elif status in ("FAILED", "KILLED", "ERROR"):
                self.alerts.append({
                    "severity": "CRITICAL",
                    "component": "Oozie",
                    "message": f"Last pipeline run {status} ({nominal_time})"
                })
            else:
                self.alerts.append({
                    "severity": "WARNING",
                    "component": "Oozie",
                    "message": f"Pipeline status: {status} ({nominal_time})"
                })

        except Exception as e:
            self.alerts.append({
                "severity": "CRITICAL",
                "component": "Oozie",
                "message": f"Cannot reach Oozie: {e}"
            })

    def check_data_freshness(self):
        """Check that yesterday's data exists in all pipeline stages."""
        dt = self.yesterday.strftime("%Y-%m-%d")

        paths_to_check = [
            (f"/data/cleaned/weblogs/dt={dt}", "Cleaned web logs"),
            (f"/data/cleaned/transactions/dt={dt}", "Cleaned transactions"),
            (f"/user/hive/warehouse/warehouse.db/fact_pageviews/dt={dt}", "Fact pageviews"),
            (f"/user/hive/warehouse/warehouse.db/fact_orders/dt={dt}", "Fact orders"),
            (f"/user/hive/warehouse/warehouse.db/agg_daily_revenue/dt={dt}", "Daily revenue"),
        ]

        for hdfs_path, description in paths_to_check:
            url = f"{self.namenode_url}/webhdfs/v1{hdfs_path}?op=GETFILESTATUS"
            try:
                req = urllib.request.Request(url)
                response = urllib.request.urlopen(req, timeout=10)
                data = json.loads(response.read().decode())
                file_status = data.get("FileStatus", {})
                if file_status.get("type") == "DIRECTORY":
                    print(f"  ✅ {description}: EXISTS ({dt})")
                else:
                    print(f"  ✅ {description}: EXISTS ({dt})")
            except urllib.error.HTTPError as e:
                if e.code == 404:
                    self.alerts.append({
                        "severity": "CRITICAL",
                        "component": "Data Freshness",
                        "message": f"MISSING: {description} for {dt}"
                    })
                else:
                    self.alerts.append({
                        "severity": "WARNING",
                        "component": "Data Freshness",
                        "message": f"Cannot check {description}: HTTP {e.code}"
                    })
            except Exception as e:
                self.alerts.append({
                    "severity": "WARNING",
                    "component": "Data Freshness",
                    "message": f"Cannot check {description}: {e}"
                })

    def check_yarn_queue_health(self):
        """Check ETL queue has resources available."""
        url = f"{self.rm_url}/ws/v1/cluster/scheduler"
        try:
            req = urllib.request.Request(url)
            req.add_header("Accept", "application/json")
            response = urllib.request.urlopen(req, timeout=10)
            data = json.loads(response.read().decode())

            # Navigate to ETL queue
            scheduler = data.get("scheduler", {}).get("schedulerInfo", {})
            etl_usage = self._find_queue_usage(scheduler, "etl")

            if etl_usage is not None:
                if etl_usage > 95:
                    self.alerts.append({
                        "severity": "WARNING",
                        "component": "YARN",
                        "message": f"ETL queue at {etl_usage:.1f}% capacity"
                    })
                else:
                    print(f"  ✅ ETL queue usage: {etl_usage:.1f}%")
        except Exception as e:
            self.alerts.append({
                "severity": "WARNING",
                "component": "YARN",
                "message": f"Cannot check YARN: {e}"
            })

    def _find_queue_usage(self, queue_data, target_name):
        """Recursively find a queue's used capacity."""
        if queue_data.get("queueName") == target_name:
            return queue_data.get("usedCapacity", 0)
        if "queues" in queue_data:
            children = queue_data["queues"].get("queue", [])
            if isinstance(children, dict):
                children = [children]
            for child in children:
                result = self._find_queue_usage(child, target_name)
                if result is not None:
                    return result
        return None

    def generate_report(self):
        """Generate health check report."""
        report = []
        report.append("")
        report.append("═" * 65)
        report.append("  PIPELINE HEALTH CHECK — " +
                       datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S"))
        report.append("═" * 65)

        if self.alerts:
            report.append("")
            report.append("  ALERTS:")
            for alert in self.alerts:
                icon = {"CRITICAL": "🔴", "WARNING": "🟡", "INFO": "🔵"}
                report.append(
                    f"  {icon.get(alert['severity'], '⚪')} "
                    f"[{alert['severity']}] {alert['component']}: "
                    f"{alert['message']}"
                )
        else:
            report.append("")
            report.append("  ✅ ALL CHECKS PASSED — Pipeline healthy")

        report.append("")
        report.append("═" * 65)
        return "\n".join(report)

    def run(self, coord_id):
        """Execute all health checks."""
        print("  Checking Oozie coordinator...")
        self.check_oozie_coordinator(coord_id)

        print("  Checking data freshness...")
        self.check_data_freshness()

        print("  Checking YARN queue health...")
        self.check_yarn_queue_health()

        report = self.generate_report()
        print(report)

        critical = sum(1 for a in self.alerts if a["severity"] == "CRITICAL")
        return 2 if critical > 0 else (1 if self.alerts else 0)
