✅ WHAT SHOULD DO

1. SIZE THE CLUSTER USING FORMULAS, NOT GUESSWORK
2. CONFIGURE RACK AWARENESS BEFORE LOADING ANY DATA
3. ENABLE ALL HA COMPONENTS BEFORE PRODUCTION TRAFFIC
4. IMPLEMENT SECURITY BEFORE GRANTING USER ACCESS
5. SET UP MONITORING BEFORE ANYTHING ELSE GOES LIVE
6. DOCUMENT EVERY OPERATIONAL PROCEDURE AS RUNBOOKS
7. ESTABLISH PERFORMANCE BASELINES ON DAY ONE
8. TEST DISASTER RECOVERY QUARTERLY
9. REVIEW CAPACITY MONTHLY
10. AUTOMATE EVERYTHING THAT RUNS MORE THAN TWICE

🚫 WHAT SHOULD NOT DO

1. DO NOT go to production without NameNode HA (Project 44)
   → Single NameNode failure = ENTIRE PLATFORM DOWN
   → This is non-negotiable for any production deployment

2. DO NOT skip rack awareness configuration
   → Without it: all 3 replicas might land on same rack
   → Rack failure = DATA LOSS (all replicas gone)
   → HDFS rack awareness is the FIRST configuration after install

3. DO NOT give all teams the same YARN queue
   → One runaway Spark job consumes all cluster resources
   → All other teams blocked for hours
   → Implement queue hierarchy with strict limits (Project 39)

4. DO NOT monitor only "is the service up"
   → Platform can be "up" but degraded beyond usability
   → Monitor: latency, throughput, queue depth, disk usage trends
   → Alert on TRENDS, not just thresholds

5. DO NOT store passwords/keytabs in HDFS or source control
   → Use credential management: HashiCorp Vault, AWS Secrets Manager
   → Rotate credentials on schedule

6. DO NOT skip capacity planning reviews
   → Data grows 2× every 12-18 months
   → If you plan for today, you're already behind

7. DO NOT allow schema-less data ingestion
   → "Dump everything as CSV" → data swamp within 6 months
   → Enforce schemas, naming conventions, partitioning standards
   → Schema registry for Avro/Parquet evolution

8. DO NOT treat the platform as "set and forget"
   → Hadoop versions, security patches, OS updates
   → Plan maintenance windows monthly
   → Test upgrades in staging BEFORE production

🚫 WHEN NOT TO USE

DO NOT USE THIS PLATFORM FOR:

1. Transactional workloads (OLTP)
   → Use PostgreSQL, MySQL, Oracle
   → Hadoop is for analytics (OLAP), not transactions

2. Sub-second interactive dashboards
   → Use Druid, ClickHouse, or Elasticsearch
   → Pre-aggregate in Hadoop, serve from specialized store

3. Real-time event streaming with millisecond latency
   → Use Kafka Streams or Apache Flink
   → Hadoop/Spark for batch + near-real-time (seconds)

4. Small datasets (<100GB total)
   → Overhead of managing Hadoop cluster unjustified
   → Use PostgreSQL + Python/Pandas

5. Graph analytics (social networks, recommendation engines)
   → Use Neo4j or Apache Spark GraphX
   → Not a primary Hadoop strength

MECHANIC 1: CLUSTER SIZING FORMULAS
═══════════════════════════════════════════════════════════

INPUTS:
  D = Current data size = 8PB (given)
  G = Daily data growth = 2TB/day (given)
  R = Retention period = 7 years (regulatory)
  RF = Replication factor = 3 (standard)
  C = Compression ratio = 0.3 (Parquet with Snappy)
  Y = Planning horizon = 3 years
  U = Target disk utilization = 70% (never fill above this)
  I = Intermediate data overhead = 1.3× (temp MapReduce/Spark data)

STORAGE CALCULATION:

  Current raw data:
    D_raw = 8PB

  Growth over planning horizon:
    D_growth = G × 365 × Y = 2TB × 365 × 3 = 2,190TB = ~2.19PB

  Total raw data at end of horizon:
    D_total = D_raw + D_growth = 8 + 2.19 = 10.19PB

  After compression:
    D_compressed = D_total × C = 10.19 × 0.3 = 3.06PB

  With replication:
    D_replicated = D_compressed × RF = 3.06 × 3 = 9.17PB

  With intermediate overhead:
    D_with_temp = D_replicated × I = 9.17 × 1.3 = 11.92PB

  At target utilization:
    D_provisioned = D_with_temp / U = 11.92 / 0.70 = 17.03PB

  Per-node disk (12 × 4TB HDDs = 48TB raw):
    Nodes_storage = D_provisioned / 48TB = 17,030 / 48 = 355 nodes

  RESULT: Need ~355 worker nodes for STORAGE over 3 years


COMPUTE CALCULATION:

  Batch window: 6 hours (10PM-4AM)
  Daily data to process: 2TB new + re-aggregate 500GB existing = 2.5TB
  Processing throughput needed: 2.5TB / 6hr = ~420GB/hr = ~117MB/s

  Per-node throughput (12 HDDs × 150MB/s × 0.5 utilization):
    Node_throughput = 12 × 150 × 0.5 = 900 MB/s

  Nodes for throughput:
    Nodes_compute = (117MB/s) / (900MB/s per node) = ~1 node (I/O)

  BUT: MapReduce/Spark CPU overhead, shuffles, joins:
    Actual throughput ~50-100 MB/s effective per node for complex ETL
    Nodes_compute_real = 2,500,000 MB / (100MB/s × 6hr × 3600s)
                       = 2,500,000 / 2,160,000 = ~2 nodes (still low)

  HOWEVER: Multiple concurrent jobs, ad-hoc queries, ML training:
    Concurrent jobs: 20 (12 teams, overlapping schedules)
    Cores per job (average): 100
    Total cores needed: 20 × 100 = 2,000 cores
    Cores per node: 24
    Nodes_concurrent = 2,000 / 24 = ~84 nodes

  Memory requirements:
    Spark executors need 12-24GB each
    20 concurrent Spark jobs × 10 executors × 16GB = 3,200GB = 3.2TB
    Nodes_memory = 3,200GB / 200GB usable per node = 16 nodes

  COMPUTE RESULT: Need ~84 nodes for concurrent compute


FINAL SIZING:

  Storage-driven: 355 nodes (DOMINATES)
  Compute-driven: 84 nodes
  
  Use storage-driven number: 355 nodes
  BUT: Round down with growth buffer
  
  INITIAL DEPLOYMENT: 200 nodes (covers 2 years)
  EXPANSION AT YEAR 2: Add 100 nodes
  EXPANSION AT YEAR 3: Add 55 nodes OR migrate cold data to S3

MECHANIC 2: RACK AWARENESS — HOW HDFS PLACES REPLICAS
═══════════════════════════════════════════════════════════

WITHOUT rack awareness (dangerous):

  Block X: replica 1 → Node 5 (Rack 4)
           replica 2 → Node 8 (Rack 4)   ← SAME RACK!
           replica 3 → Node 12 (Rack 4)  ← SAME RACK!

  If Rack 4 loses power → ALL 3 replicas GONE → DATA LOSS

WITH rack awareness (correct):

  Block X: replica 1 → Node 5 (Rack 4)   ← Local rack
           replica 2 → Node 55 (Rack 5)  ← Different rack
           replica 3 → Node 58 (Rack 5)  ← Same as replica 2's rack

  HDFS Rack Placement Policy:
  1. First replica: same node as writer (or random if remote client)
  2. Second replica: DIFFERENT rack from first
  3. Third replica: SAME rack as second but DIFFERENT node

  WHY this policy:
  → Survives single rack failure (replicas on 2 racks)
  → Minimizes cross-rack writes (2 of 3 replicas on same rack)
  → Balances reliability vs. network bandwidth

  IMPLEMENTATION:
  → Rack topology script tells NameNode which rack each node belongs to
  → NameNode uses this mapping for EVERY block placement decision

#!/bin/bash
# Rack topology script for HDFS rack awareness
# Maps IP addresses/hostnames to rack identifiers
#
# Called by NameNode for EVERY block placement decision
# Input: list of IP addresses (one per argument)
# Output: rack path for each IP (one per line)

HADOOP_CONF=/opt/hadoop/etc/hadoop

while [ $# -gt 0 ] ; do
    nodeArg=$1
    result=""
    
    # Look up rack assignment from topology data file
    if [ -f ${HADOOP_CONF}/rack-topology.data ] ; then
        result=$(grep -w "${nodeArg}" ${HADOOP_CONF}/rack-topology.data | awk '{print $2}')
    fi
    
    # Default rack if not found (DANGEROUS — should alert on this)
    if [ -z "$result" ] ; then
        result="/default-rack"
        echo "WARNING: No rack mapping for ${nodeArg}, using default" >&2
    fi
    
    echo "$result"
    shift
done
TOPOLOGY_SCRIPT

chmod +x /opt/hadoop/etc/hadoop/rack-topology.sh

class AlertLevel:
    OK = "OK"
    WARNING = "WARNING"
    CRITICAL = "CRITICAL"


class Alert:
    def __init__(self, component, check_name, level, message, value=None, threshold=None):
        self.component = component
        self.check_name = check_name
        self.level = level
        self.message = message
        self.value = value
        self.threshold = threshold
        self.timestamp = datetime.datetime.now().isoformat()

    def __str__(self):
        icon = "✅" if self.level == AlertLevel.OK else (
            "⚠️" if self.level == AlertLevel.WARNING else "🔴"
        )
        detail = ""
        if self.value is not None and self.threshold is not None:
            detail = f" (value={self.value}, threshold={self.threshold})"
        return f"{icon} [{self.level}] {self.component}/{self.check_name}: {self.message}{detail}"

def http_get_json(url, timeout=10):
    """Fetch JSON from HTTP endpoint using curl."""
    try:
        result = subprocess.run(
            ["curl", "-s", "--connect-timeout", str(timeout), url],
            capture_output=True, text=True, timeout=timeout + 5
        )
        if result.returncode == 0 and result.stdout.strip():
            return json.loads(result.stdout)
    except (subprocess.TimeoutExpired, json.JSONDecodeError, Exception) as e:
        return None
    return None

def check_hdfs_health(config):
    """
    Checks all critical HDFS metrics.
    References: Project 2 (HDFS fundamentals), Project 44 (HA)
    """
    alerts = []
    thresholds = config["thresholds"]
    nn_url = config["namenode_url"]

    # ── 1A: HDFS JMX metrics ──
    jmx_url = f"{nn_url}/jmx?qry=Hadoop:service=NameNode,name=FSNamesystemState"
    data = http_get_json(jmx_url)

    if data is None:
        alerts.append(Alert("HDFS", "namenode_reachable", AlertLevel.CRITICAL,
                            "Cannot reach NameNode JMX endpoint"))
        return alerts

    try:
        beans = data.get("beans", [{}])
        metrics = beans[0] if beans else {}

        # Capacity check
        capacity_total = metrics.get("CapacityTotal", 1)
        capacity_used = metrics.get("CapacityUsed", 0)
        capacity_pct = (capacity_used / capacity_total * 100) if capacity_total > 0 else 0

        if capacity_pct >= thresholds["hdfs_capacity_critical_pct"]:
            alerts.append(Alert("HDFS", "capacity", AlertLevel.CRITICAL,
                                f"HDFS capacity at {capacity_pct:.1f}%",
                                capacity_pct, thresholds["hdfs_capacity_critical_pct"]))
        elif capacity_pct >= thresholds["hdfs_capacity_warning_pct"]:
            alerts.append(Alert("HDFS", "capacity", AlertLevel.WARNING,
                                f"HDFS capacity at {capacity_pct:.1f}%",
                                capacity_pct, thresholds["hdfs_capacity_warning_pct"]))
        else:
            alerts.append(Alert("HDFS", "capacity", AlertLevel.OK,
                                f"HDFS capacity at {capacity_pct:.1f}%"))

        # Under-replicated blocks
        under_replicated = metrics.get("UnderReplicatedBlocks", 0)

        if under_replicated >= thresholds["hdfs_under_replicated_critical"]:
            alerts.append(Alert("HDFS", "under_replicated", AlertLevel.CRITICAL,
                                f"{under_replicated} under-replicated blocks",
                                under_replicated, thresholds["hdfs_under_replicated_critical"]))
        elif under_replicated >= thresholds["hdfs_under_replicated_warning"]:
            alerts.append(Alert("HDFS", "under_replicated", AlertLevel.WARNING,
                                f"{under_replicated} under-replicated blocks",
                                under_replicated, thresholds["hdfs_under_replicated_warning"]))
        else:
            alerts.append(Alert("HDFS", "under_replicated", AlertLevel.OK,
                                f"{under_replicated} under-replicated blocks"))

        # Missing blocks (CRITICAL — potential data loss!)
        missing_blocks = metrics.get("MissingBlocks", 0)

        if missing_blocks >= thresholds["hdfs_missing_blocks_critical"]:
            alerts.append(Alert("HDFS", "missing_blocks", AlertLevel.CRITICAL,
                                f"MISSING BLOCKS: {missing_blocks} — POTENTIAL DATA LOSS",
                                missing_blocks, thresholds["hdfs_missing_blocks_critical"]))
        else:
            alerts.append(Alert("HDFS", "missing_blocks", AlertLevel.OK,
                                f"No missing blocks"))

        # Dead DataNodes
        live_nodes = metrics.get("NumLiveDataNodes", 0)
        dead_nodes = metrics.get("NumDeadDataNodes", 0)

        if dead_nodes >= thresholds["hdfs_dead_datanodes_critical"]:
            alerts.append(Alert("HDFS", "dead_datanodes", AlertLevel.CRITICAL,
                                f"{dead_nodes} dead DataNodes (live: {live_nodes})",
                                dead_nodes, thresholds["hdfs_dead_datanodes_critical"]))
        elif dead_nodes >= thresholds["hdfs_dead_datanodes_warning"]:
            alerts.append(Alert("HDFS", "dead_datanodes", AlertLevel.WARNING,
                                f"{dead_nodes} dead DataNodes (live: {live_nodes})",
                                dead_nodes, thresholds["hdfs_dead_datanodes_warning"]))
        else:
            alerts.append(Alert("HDFS", "dead_datanodes", AlertLevel.OK,
                                f"All {live_nodes} DataNodes alive"))

        # NameNode heap usage
        nn_heap_url = f"{nn_url}/jmx?qry=java.lang:type=Memory"
        heap_data = http_get_json(nn_heap_url)
        if heap_data and heap_data.get("beans"):
            heap_metrics = heap_data["beans"][0]
            heap_used = heap_metrics.get("HeapMemoryUsage", {}).get("used", 0)
            heap_max = heap_metrics.get("HeapMemoryUsage", {}).get("max", 1)
            heap_pct = (heap_used / heap_max * 100) if heap_max > 0 else 0

            if heap_pct >= thresholds["namenode_heap_critical_pct"]:
                alerts.append(Alert("HDFS", "namenode_heap", AlertLevel.CRITICAL,
                                    f"NameNode heap at {heap_pct:.1f}%",
                                    heap_pct, thresholds["namenode_heap_critical_pct"]))
            elif heap_pct >= thresholds["namenode_heap_warning_pct"]:
                alerts.append(Alert("HDFS", "namenode_heap", AlertLevel.WARNING,
                                    f"NameNode heap at {heap_pct:.1f}%",
                                    heap_pct, thresholds["namenode_heap_warning_pct"]))
            else:
                alerts.append(Alert("HDFS", "namenode_heap", AlertLevel.OK,
                                    f"NameNode heap at {heap_pct:.1f}%"))

    except (KeyError, IndexError, TypeError) as e:
        alerts.append(Alert("HDFS", "parsing", AlertLevel.WARNING,
                            f"Could not parse HDFS metrics: {e}"))

    return alerts

def check_pipeline_slas(config):
    """
    Check if critical pipelines completed within SLA windows.
    References: Project 31 (Oozie orchestration)
    """
    alerts = []
    thresholds = config["thresholds"]
    rm_url = config["resourcemanager_url"]
    now = datetime.datetime.now()

    # Check if we're past the batch window
    batch_end_hour = thresholds["batch_window_end_hour"]

    if now.hour >= batch_end_hour and now.hour < 12:
        # We should check if nightly batch completed
        apps_url = (f"{rm_url}/ws/v1/cluster/apps?"
                    f"states=FINISHED,FAILED,KILLED&"
                    f"startedTimeBegin={int((now - datetime.timedelta(hours=12)).timestamp() * 1000)}")

        data = http_get_json(apps_url)

        if data and data.get("apps", {}).get("app"):
            apps = data["apps"]["app"]

            # Look for nightly batch pipeline
            batch_apps = [a for a in apps if "LogisticsPipeline" in a.get("name", "")
                          or "NightlyETL" in a.get("name", "")]

            if not batch_apps:
                alerts.append(Alert("PIPELINE", "nightly_batch", AlertLevel.CRITICAL,
                                    "Nightly batch pipeline NOT FOUND in completed apps"))
            else:
                latest = max(batch_apps, key=lambda a: a.get("finishedTime", 0))
                state = latest.get("finalStatus", "UNKNOWN")
                elapsed_ms = latest.get("elapsedTime", 0)
                elapsed_hours = elapsed_ms / (1000 * 3600)

                if state != "SUCCEEDED":
                    alerts.append(Alert("PIPELINE", "nightly_batch", AlertLevel.CRITICAL,
                                        f"Nightly batch FAILED with status: {state}"))
                elif elapsed_hours > thresholds["batch_max_duration_hours"]:
                    alerts.append(Alert("PIPELINE", "nightly_batch", AlertLevel.WARNING,
                                        f"Nightly batch took {elapsed_hours:.1f}h "
                                        f"(SLA: {thresholds['batch_max_duration_hours']}h)",
                                        elapsed_hours, thresholds["batch_max_duration_hours"]))
                else:
                    alerts.append(Alert("PIPELINE", "nightly_batch", AlertLevel.OK,
                                        f"Nightly batch completed in {elapsed_hours:.1f}h"))
        else:
            alerts.append(Alert("PIPELINE", "nightly_batch", AlertLevel.WARNING,
                                "Could not retrieve completed applications"))

    return alerts

                        alerts.append(Alert("DISK", f"mount_{mount}",

                                            AlertLevel.WARNING,
                                            f"Disk {mount} at {usage_pct}%",
                                            usage_pct,
                                            thresholds["disk_usage_warning_pct"]))

    except Exception as e:
        alerts.append(Alert("DISK", "check", AlertLevel.WARNING,
                            f"Could not check disk: {e}"))

    if not any(a.level != AlertLevel.OK for a in alerts):
        alerts.append(Alert("DISK", "all_mounts", AlertLevel.OK,
                            "All disk mounts within limits"))

    return alerts

def check_zookeeper_health(config):
    """
    Verify ZooKeeper quorum is healthy.
    ZK is the foundation for ALL HA — if ZK fails, everything fails.
    """
    alerts = []
    zk_hosts = ["mgmt-01:2181", "mgmt-02:2181", "mgmt-03:2181"]

    live_count = 0
    leader_count = 0

    for host in zk_hosts:
        hostname, port = host.split(":")
        try:
            result = subprocess.run(
                ["bash", "-c", f"echo ruok | nc -w 3 {hostname} {port}"],
                capture_output=True, text=True, timeout=5
            )
            if "imok" in result.stdout:
                live_count += 1

            # Check role
            result2 = subprocess.run(
                ["bash", "-c", f"echo stat | nc -w 3 {hostname} {port}"],
                capture_output=True, text=True, timeout=5
            )
            if "leader" in result2.stdout.lower():
                leader_count += 1

        except Exception:
            pass

    quorum_healthy = live_count >= 2  # Majority of 3
    has_leader = leader_count == 1

    if not quorum_healthy:
        alerts.append(Alert("ZOOKEEPER", "quorum", AlertLevel.CRITICAL,
                            f"ZK quorum LOST: only {live_count}/3 alive "
                            f"— ALL HA DISABLED",
                            live_count, 2))
    elif live_count < 3:
        alerts.append(Alert("ZOOKEEPER", "quorum", AlertLevel.WARNING,
                            f"ZK degraded: {live_count}/3 alive "
                            f"— one more failure = quorum loss",
                            live_count, 3))
    else:
        alerts.append(Alert("ZOOKEEPER", "quorum", AlertLevel.OK,
                            f"All 3 ZK nodes alive, leader elected: {has_leader}"))

    return alerts

def dispatch_alerts(alerts, config):
    """
    Send alerts to appropriate channels based on severity.
    
    OK       → Log only
    WARNING  → Log + Email
    CRITICAL → Log + Email + PagerDuty
    """

    log_dir = config["log_dir"]
    os.makedirs(log_dir, exist_ok=True)

    timestamp = datetime.datetime.now().strftime("%Y%m%d_%H%M%S")
    log_file = os.path.join(log_dir, f"monitor_{timestamp}.log")

    critical_alerts = [a for a in alerts if a.level == AlertLevel.CRITICAL]
    warning_alerts = [a for a in alerts if a.level == AlertLevel.WARNING]
    ok_alerts = [a for a in alerts if a.level == AlertLevel.OK]

    # ── Write log file ──
    with open(log_file, "w") as f:
        f.write(f"Platform Health Check — {timestamp}\n")
        f.write("=" * 70 + "\n\n")

        f.write(f"CRITICAL: {len(critical_alerts)}\n")
        f.write(f"WARNING:  {len(warning_alerts)}\n")
        f.write(f"OK:       {len(ok_alerts)}\n\n")

        for alert in alerts:
            f.write(f"{alert}\n")

    # ── Console output ──
    print(f"\n{'=' * 70}")
    print(f"PLATFORM HEALTH CHECK — {timestamp}")
    print(f"{'=' * 70}")
    print(f"\n  🔴 CRITICAL: {len(critical_alerts)}")
    print(f"  ⚠️  WARNING:  {len(warning_alerts)}")
    print(f"  ✅ OK:       {len(ok_alerts)}")
    print(f"\n{'─' * 70}")

    for alert in critical_alerts:
        print(f"  {alert}")
    for alert in warning_alerts:
        print(f"  {alert}")
    for alert in ok_alerts:
        print(f"  {alert}")

    print(f"\n{'─' * 70}")
    print(f"  Log written to: {log_file}")

    # ── Send email for warnings ──
    if warning_alerts or critical_alerts:
        subject_prefix = "🔴 CRITICAL" if critical_alerts else "⚠️ WARNING"
        subject = f"{subject_prefix}: Platform Health Alert"
        body = "\n".join(str(a) for a in critical_alerts + warning_alerts)

import subprocess

class DisasterRecoveryManager:
    """
    Manages HDFS data replication to DR site and S3 archive.
    
    DR Strategy:
    → Primary: On-premise HDFS (200 nodes)
    → DR: Remote datacenter HDFS (50 nodes, can scale)
    → Archive: S3 Glacier (unlimited, cheap)
    
    RPO (Recovery Point Objective): 24 hours
    → Daily replication means max 24 hours of data loss
    
    RTO (Recovery Time Objective): 4 hours
    → Time to activate DR site and resume processing
    
    References: All storage projects (2, 14, 30, 43, 44, 49)
    """

    def __init__(self):
        self.primary_nn = "hdfs://company-hdfs"
        self.dr_nn = "hdfs://company-hdfs-dr"
        self.s3_archive = "s3a://company-dr-archive"
        self.log_dir = "/var/log/disaster-recovery/"
        self.timestamp = datetime.datetime.now().strftime("%Y%m%d_%H%M%S")
        os.makedirs(self.log_dir, exist_ok=True)

    def replicate_critical_data(self):
        """
        Replicate critical datasets to DR site using DistCp.
        DistCp uses MapReduce internally for parallel copy.
        """

        print("=" * 70)
        print(f"DISASTER RECOVERY REPLICATION — {self.timestamp}")
        print("=" * 70)

        # Define critical paths to replicate
        critical_paths = [
            {
                "source": f"{self.primary_nn}/data/logistics/",
                "target": f"{self.dr_nn}/data/logistics/",
                "description": "Logistics pipeline data",
                "priority": "P1"
            },
            {
                "source": f"{self.primary_nn}/data/claims/",
                "target": f"{self.dr_nn}/data/claims/",
                "description": "Claims processing data",
                "priority": "P1"
            },
            {
                "source": f"{self.primary_nn}/data/warehouse/",
                "target": f"{self.dr_nn}/data/warehouse/",
                "description": "Data warehouse (Hive tables)",
                "priority": "P1"
            },
            {
                "source": f"{self.primary_nn}/data/ml-models/",
                "target": f"{self.dr_nn}/data/ml-models/",
                "description": "ML model artifacts",
                "priority": "P2"
            },
            {
                "source": f"{self.primary_nn}/user/",
                "target": f"{self.dr_nn}/user/",
                "description": "User home directories",
                "priority": "P3"
            }
        ]

        results = []

        for path_config in critical_paths:
            source = path_config["source"]
            target = path_config["target"]
            desc = path_config["description"]
            priority = path_config["priority"]

            print(f"\n[{priority}] Replicating: {desc}")
            print(f"  Source: {source}")
            print(f"  Target: {target}")

            start_time = datetime.datetime.now()

            try:
                # DistCp with -update flag: only copy new/changed files
                cmd = [
                    "hadoop", "distcp",
                    "-update",           # Skip unchanged files
                    "-skipcrccheck",     # Skip CRC (faster, rely on size/mtime)
                    "-m", "20",          # 20 parallel map tasks
                    "-bandwidth", "50",  # Limit to 50MB/s per mapper
                    "-strategy", "dynamic",
                    "-log", f"{self.primary_nn}/tmp/distcp-logs/{self.timestamp}/",
                    source, target
                ]

                result = subprocess.run(
                    cmd,
                    capture_output=True, text=True,
                    timeout=7200  # 2 hour timeout per path
                )

                elapsed = (datetime.datetime.now() - start_time).total_seconds()

                if result.returncode == 0:
                    print(f"  ✅ Completed in {elapsed:.0f}s")
                    results.append((desc, "SUCCESS", elapsed))
                else:
                    print(f"  ❌ FAILED: {result.stderr[:200]}")
                    results.append((desc, "FAILED", elapsed))

            except subprocess.TimeoutExpired:
                print(f"  ❌ TIMEOUT after 2 hours")
                results.append((desc, "TIMEOUT", 7200))
            except Exception as e:
                print(f"  ❌ ERROR: {e}")
                results.append((desc, "ERROR", 0))

        return results

    def archive_to_s3(self):
        """
        Archive old data to S3 Glacier for long-term retention.
        Data older than 1 year moves to S3.
        """

        print(f"\n{'=' * 70}")
        print("ARCHIVAL: Moving old data to S3 Glacier")
        print(f"{'=' * 70}")

        one_year_ago = (datetime.datetime.now() -
                        datetime.timedelta(days=365)).strftime("%Y-%m-%d")

        archive_paths = [
            f"{self.primary_nn}/data/logistics/raw/",
            f"{self.primary_nn}/data/claims/raw/",
        ]

        for path in archive_paths:
            print(f"\n  Archiving data older than {one_year_ago} from {path}")

            # In production: use Spark to identify and move old partitions
            # Here: demonstrate the DistCp approach
            try:
                cmd = [
                    "hadoop", "distcp",
                    "-update",
                    "-m", "10",
                    "-bandwidth", "100",
                    path,
                    f"{self.s3_archive}/archive/"
                ]

                result = subprocess.run(
                    cmd, capture_output=True, text=True, timeout=3600
                )

                if result.returncode == 0:
                    print(f"  ✅ Archived successfully")
                    # After verification: delete from HDFS to reclaim space
                    # hdfs dfs -rm -r {path}/year=2022/
                else:
                    print(f"  ❌ Archive failed: {result.stderr[:200]}")

            except Exception as e:
                print(f"  ❌ Error: {e}")

    def validate_replication(self):
        """
        Validate DR site has correct data by comparing file counts and sizes.
        """

        print(f"\n{'=' * 70}")
        print("DR VALIDATION: Comparing primary vs DR site")
        print(f"{'=' * 70}")

        validation_paths = [
            "/data/logistics/",
            "/data/warehouse/",
        ]

        all_valid = True

        for path in validation_paths:
            print(f"\n  Validating: {path}")

            try:
                # Primary count
                primary_result = subprocess.run(
                    ["hdfs", "dfs", "-count", f"{self.primary_nn}{path}"],
                    capture_output=True, text=True, timeout=60
                )
                primary_parts = primary_result.stdout.strip().split()
                primary_files = int(primary_parts[1]) if len(primary_parts) >= 3 else -1
                primary_bytes = int(primary_parts[2]) if len(primary_parts) >= 3 else -1

                # DR count
                dr_result = subprocess.run(
                    ["hdfs", "dfs", "-count", f"{self.dr_nn}{path}"],
                    capture_output=True, text=True, timeout=60
                )
                dr_parts = dr_result.stdout.strip().split()
                dr_files = int(dr_parts[1]) if len(dr_parts) >= 3 else -1
                dr_bytes = int(dr_parts[2]) if len(dr_parts) >= 3 else -1

                files_match = primary_files == dr_files
                # Allow 1% size difference (compression variations)
                bytes_close = abs(primary_bytes - dr_bytes) < primary_bytes * 0.01

                status = "✅" if (files_match and bytes_close) else "❌"
                if not (files_match and bytes_close):
                    all_valid = False

                print(f"    Primary: {primary_files} files, "
                      f"{primary_bytes / (1024**3):.1f} GB")
                print(f"    DR:      {dr_files} files, "
                      f"{dr_bytes / (1024**3):.1f} GB")
                print(f"    Status:  {status}")

            except Exception as e:
                print(f"    ❌ Validation error: {e}")
                all_valid = False

        return all_valid

    def generate_dr_report(self, replication_results, validation_ok):
        """Generate DR status report."""

        report = {
            "timestamp": self.timestamp,
            "replication_results": [
                {"dataset": desc, "status": status, "duration_seconds": dur}
                for desc, status, dur in replication_results
            ],
            "validation_passed": validation_ok,
            "rpo_met": all(s == "SUCCESS" for _, s, _ in replication_results),
            "next_scheduled": (datetime.datetime.now() +
                               datetime.timedelta(hours=24)).isoformat()
        }

        report_path = os.path.join(self.log_dir, f"dr_report_{self.timestamp}.json")
        with open(report_path, "w") as f:
            json.dump(report, f, indent=2)

        print(f"\n{'=' * 70}")
        print("DR REPORT SUMMARY")
        print(f"{'=' * 70}")

        for desc, status, dur in replication_results:
            icon = "✅" if status == "SUCCESS" else "❌"
            print(f"  {icon} {desc}: {status} ({dur:.0f}s)")

        rpo_met = report["rpo_met"]
        print(f"\n  RPO (24h) met: {'✅ YES' if rpo_met else '❌ NO'}")
        print(f"  Validation:    {'✅ PASSED' if validation_ok else '❌ FAILED'}")
        print(f"  Report: {report_path}")

        if not rpo_met:
            print(f"\n  🔴 ALERT: RPO NOT MET — investigate replication failures")
            print(f"     ACTION: Manual replication required before next business day")

        return report


def main():
    dr = DisasterRecoveryManager()

    # Step 1: Replicate critical data
    results = dr.replicate_critical_data()

    # Step 2: Archive old data to S3
    dr.archive_to_s3()

    # Step 3: Validate DR site
    validation_ok = dr.validate_replication()

    # Step 4: Generate report
    report = dr.generate_dr_report(results, validation_ok)

    # Exit with error if DR failed
    if not report["rpo_met"]:
        sys.exit(1)


if __name__ == "__main__":
    main()

from pyspark.sql import SparkSession

class PerformanceBaseline:
    """
    Establishes and monitors performance baselines for critical pipelines.
    Detects gradual degradation before it becomes a crisis.
    
    Pattern:
    1. Run benchmark queries/jobs
    2. Record execution time
    3. Compare to historical baseline
    4. Alert if regression exceeds threshold
    """

    def __init__(self, baseline_dir="/var/lib/platform/baselines/"):
        self.baseline_dir = baseline_dir
        os.makedirs(baseline_dir, exist_ok=True)
        self.timestamp = datetime.datetime.now().isoformat()
        self.results = {}

    def run_benchmark(self, spark, name, query_func):
        """Run a benchmark function and record timing."""
        print(f"\n  Running benchmark: {name}")

        import time
        start = time.time()

        try:
            result = query_func(spark)
            elapsed = time.time() - start
            self.results[name] = {
                "elapsed_seconds": elapsed,
                "status": "SUCCESS",
                "timestamp": self.timestamp,
                "result_summary": str(result)[:200]
            }
            print(f"    ✅ Completed in {elapsed:.2f}s")
        except Exception as e:
            elapsed = time.time() - start
            self.results[name] = {
                "elapsed_seconds": elapsed,
                "status": "FAILED",
                "timestamp": self.timestamp,
                "error": str(e)[:200]
            }
            print(f"    ❌ Failed after {elapsed:.2f}s: {e}")

        return self.results[name]

    def compare_to_baseline(self, regression_threshold=1.5):
        """
        Compare current results to historical baseline.
        Alert if any benchmark exceeds threshold × baseline.
        """
        baseline_file = os.path.join(self.baseline_dir, "baseline.json")
        history_file = os.path.join(self.baseline_dir, "history.json")

        # Load existing baseline
        baseline = {}
        if os.path.exists(baseline_file):
            with open(baseline_file, "r") as f:
                baseline = json.load(f)

        # Load history
        history = []
        if os.path.exists(history_file):
            with open(history_file, "r") as f:
                history = json.load(f)

        # Append current results to history
        history.append({
            "timestamp": self.timestamp,
            "results": self.results
        })

        # Keep last 52 weeks of history
        history = history[-52:]

        with open(history_file, "w") as f:
            json.dump(history, f, indent=2)

        # Compare
        regressions = []
        print(f"\n{'=' * 70}")
        print("PERFORMANCE REGRESSION ANALYSIS")
        print(f"{'=' * 70}")
        print(f"\n  {'Benchmark':<35} {'Current':>10} {'Baseline':>10} {'Ratio':>8} {'Status':>8}")
        print(f"  {'─' * 35} {'─' * 10} {'─' * 10} {'─' * 8} {'─' * 8}")

        for name, result in self.results.items():
            if result["status"] != "SUCCESS":
                print(f"  {name:<35} {'FAILED':>10} {'N/A':>10} {'N/A':>8} {'❌':>8}")
                regressions.append((name, "FAILED", 0, 0))
                continue

            current = result["elapsed_seconds"]

            if name in baseline:
                base_time = baseline[name]["elapsed_seconds"]
                ratio = current / base_time if base_time > 0 else 999

                if ratio > regression_threshold:
                    status = "🔴 REG"
                    regressions.append((name, "REGRESSION", current, base_time))
                elif ratio > 1.2:
                    status = "⚠️ SLOW"
                else:
                    status = "✅ OK"

                print(f"  {name:<35} {current:>9.2f}s {base_time:>9.2f}s "
                      f"{ratio:>7.2f}x {status:>8}")
            else:
                # First run — establish baseline
                baseline[name] = {
                    "elapsed_seconds": current,
                    "established": self.timestamp
                }
                print(f"  {name:<35} {current:>9.2f}s {'NEW':>10} {'N/A':>8} {'📊 BASE':>8}")

        # Update baseline (rolling average of last 4 runs)
        if len(history) >= 4:
            for name in self.results:
                recent_times = []
                for h in history[-4:]:
                    if name in h.get("results", {}):
                        r = h["results"][name]
                        if r.get("status") == "SUCCESS":
                            recent_times.append(r["elapsed_seconds"])

                if recent_times:
                    baseline[name] = {
                        "elapsed_seconds": sum(recent_times) / len(recent_times),
                        "established": self.timestamp,
                        "sample_count": len(recent_times)
                    }

        with open(baseline_file, "w") as f:
            json.dump(baseline, f, indent=2)

        print(f"\n  Baseline file: {baseline_file}")
        print(f"  History file:  {history_file}")
        print(f"  History depth: {len(history)} runs")

        if regressions:
            print(f"\n  🔴 REGRESSIONS DETECTED: {len(regressions)}")
            for name, status, current, base in regressions:
                if status == "FAILED":
                    print(f"    → {name}: FAILED (was working)")
                else:
                    print(f"    → {name}: {current:.2f}s vs baseline {base:.2f}s "
                          f"({current/base:.1f}x slower)")
            print(f"\n  ACTIONS:")
            print(f"    1. Check for data skew changes (Project 33)")
            print(f"    2. Check for small files accumulation (Project 2)")
            print(f"    3. Check HDFS block distribution (hdfs fsck)")
            print(f"    4. Check YARN queue contention (Project 39)")
            print(f"    5. Profile slowest stage in Spark UI (Project 48)")

        return regressions


def define_benchmarks():
    """Define the benchmark suite."""

    def benchmark_full_table_scan(spark):
        """Baseline: How long to scan entire shipments table."""
        df = spark.read.parquet("hdfs:///data/logistics/spark_output/enriched_partitioned/")
        count = df.count()
        return f"count={count}"

    def benchmark_filtered_scan(spark):
        """Baseline: How long to scan with partition pruning."""
        df = spark.read.parquet("hdfs:///data/logistics/spark_output/enriched_partitioned/")
        count = df.filter(F.col("region") == "NORTH").count()
        return f"count={count}"

    def benchmark_aggregation(spark):
        """Baseline: How long to aggregate by region."""
        df = spark.read.parquet("hdfs:///data/logistics/spark_output/enriched_partitioned/")
        result = df.groupBy("region").agg(
            F.count("*").alias("cnt"),
            F.sum("cost").alias("total")
        ).collect()
        return f"regions={len(result)}"

    def benchmark_join(spark):
        """Baseline: How long to join shipments with customers."""
        shipments = spark.read.parquet(
            "hdfs:///data/logistics/spark_output/enriched_partitioned/"
        )
        customers = spark.read.parquet(
            "hdfs:///data/logistics/customers_parquet/"
        ) if os.path.exists("/tmp/customers_exists") else spark.read \
            .option("header", "true") \
            .csv("hdfs:///data/logistics/customers/customers.csv")

        count = shipments.join(F.broadcast(customers), "customer_id").count()
        return f"joined_count={count}"

    def benchmark_complex_pipeline(spark):
        """Baseline: Full pipeline end-to-end."""
        df = spark.read.parquet("hdfs:///data/logistics/spark_output/enriched_partitioned/")

        result = df \
            .filter(F.col("cost") > 0) \
            .groupBy("region", "status") \
            .agg(
                F.count("*").alias("cnt"),
                F.avg("cost").alias("avg_cost"),
                F.countDistinct("customer_id").alias("unique_custs")
            ) \
            .orderBy(F.desc("cnt")) \
            .collect()
        return f"result_rows={len(result)}"

    return {
        "full_table_scan": benchmark_full_table_scan,
        "filtered_scan_partition_prune": benchmark_filtered_scan,
        "region_aggregation": benchmark_aggregation,
        "broadcast_join": benchmark_join,
        "complex_pipeline": benchmark_complex_pipeline,
    }

        }, f, indent=2)
